## テーマ

**詳細は `00_シリーズ共通/01_テーマ/三部のテーマ軸.md` がマスター。**

### シリーズ全体

超常の力でできたシステムを解体し、**ただの人間（竹造）が生きられる世界**を残す。

### 三部のテーマ（要約）

| 部 | テーマ |
|----|--------|
| **第1部** | 自由と孤独の楽園は、大きな組織に勝てない。**敗北・支援・恭順**——生き延びるための現実 |
| **第2部** | 他者を利用してだますな。**生身の怒り**と**泥臭い小さな勝利**が、完璧なシステムを狂わす |
| **第3部** | 力を持った言仁が超常を消し、**竹造（ただの人間）に世界を明け渡す** |

### 旧・テーマメモ（参照用）

- 天孫降臨OS＝「天上の額（しゃく）」。人間の手から外す。
- 第1部着地：**「お前はお前の国を持て」**（母の言葉）
- 問い：真実の愛は存在するか／神を殺すことは救いか／自分の国とは何か

## 物部流（呪術の芯）

> **たたき台正本** → `00_シリーズ共通/物部流/02_物部流再定義.md`

### 身体（健康先行）

基本：足裏・**腹式呼吸**・背筋・海の音に吐息を合わせる（潮合）。

**例として残す型**（必須ではない）：首を伸ばし肩を下ろす → 脇を締め → 指先の変化（小指の痺れも合図の一つ）。

### 言霊の芯：掛詞・ダブルミーニング

物部流は「有名な祝詞の引用」ではない。**やりたいこと**が深層に立ち、表層は祝詞・本の音を借りて**言いなおす**。詳細 → `02_物部流再定義.md`

### 言霊の根：布瑠の言（反魂）

```
ひふみよ　いむなや　ここのたり
ふるべ　ゆらゆらと　ふるべ
```

**深層**：「**古き魂が、ゆらゆらと降るところ**」——反魂専用。銛には使わない。

### 銛強化（第1部・尖刺）

```
ひ　ふ　み
ふる、尖（とが）れ、降（ふ）れ
ゆら、止（と）め
と　刺（さ）せ
```

**深層**：古き力を尖らせ降ろし、揺らぎを止めて刺す。

- **八重垣** … 天孫系の整文祝詞（参考のみ）。物部流の掛詞体系とは別。
