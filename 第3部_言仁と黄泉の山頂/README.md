## 言仁の結末（確定）

- **死なない**。自ら**神器たるしるし**を抜き、**人として生きる**（平家物語的「安徳の死」で締めない）。  
- 決戦後、**宇都宮**という看板は剥落。言仁は**歴史から消える**（隠棲）。  
- 詳細 → **`設定/01_言仁の結末と神器たるしるし.md`**

## 宇都宮の結末
- 朝綱：必要なら短く——空けた殻を他人に預けた報い。
- **竹崎**が肥後に「自分の国」。第1部母の言葉の回収。
