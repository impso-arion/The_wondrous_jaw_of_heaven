### 三人の「載り」の違い

| 誰 | 載りの型 |
|----|----------|
| **言仁** | 生まれながら**「載り＋人」**。血に**神器たるしるし**（正規の署名キー）。何もせずとも国に繋がっている |
| **木花** | **依代の人**。もとは**祈れば載る**。神気漏れ後は**祈らずとも載る**（→ `02_木花.md`） |
| **竹崎** | ただの人。**頑張って祝詞を唱えれば載る**が、**呪力はない**。第3部の主人公 |

### 天逆鉾と神気の潮流

| 項目 | 内容 |
|------|------|
| **天逆鉾** | 霧島に立つ**栓**。天からの**神気の流入を止めている**杭・ゴムパッキン |
| **悪の手（崇徳）** | 逆鉾を**抜く**＝霧島の神気ポートを**開ける** |
| **神気の潮流** | 抜くと、汚染が**流れ込む**。空中ではなく、山の**水脈・霧・雨・潮の目**を伝う（魔力＝潮を伝う原則どおり） |
| **結果** | **依代の人**が、祈らずとも言仁と同じ**「載り＋人」**になれてしまう。崇徳はこれで**新しい国**を作る |

### 摘出の意味（更新）★

- 旧理解：体内のしるしを**捨てる**だけ。  
- **新理解**：言仁は**新しい逆鉾**を立て、自分から剥がした**載り（しるし）**を、その逆鉾の**封止材（ゴムパッキン・接着剤）**にする。  
- これで**神気の潮流が止まる**＝霧島の穴が塞がる＝超常が消える。  
- **死ではない**。載っている部分を**外して栓に使う**。言仁は**人**として残る。

## ラストバトルへの道

> 崇徳側の計画 → `03_崇徳院.md`（古墳東征・霧島再降臨・山頂の罠・ニニギ）

### 1. 疲弊と叫び

- 崇徳に向け：**「お前の国は、ここにあるぞ！　取りに来い！　崇徳！」**  
- 国＝**ここ**（山・載り・標的）——講義ではなく**挑発の結節**。崇徳には**罠**。

### 2. 載りを外した状態

> `../../設定/01_言仁の結末と神器たるしるし.md`・`04_載りとは.md`

| 項目 | 内容 |
|------|------|
| **何を外す** | 体内の**載り**（神器たるしるし・天子のバッファ） |
| **どう使う** | 捨てるのではなく、**新しい逆鉾の封止材**にする（霧島の穴を塞ぐゴムパッキン） |
| **見え方** | 肩の荷が下りる。同い年の顔。祝詞の**効き**が止まる体感 |
| **死** | **死ぬために戦わない**。載っている部分だけ外して**栓に使う** |

### 3. 三人の作業

| 役 | 行動 |
|----|------|
| **言仁** | 唱える。自分の**載りを抜く**。それを逆鉾に**封じる**。説明は最短 |
| **竹崎** | 押さえる。人間の手。**新しい逆鉾を立てる／旗を動かす** |
| **木花** | **載りの上流**を身体で押さえる。依代の同期解除。前衛として**そろう** |
| **共同** | **分霊アイテム**（形代）を破壊——神気流入・再降臨の経路を断つ |
| **天の逆鉾** | **新しく立てる**。**刃が天を向く**——降臨を**刺し止める**栓。言仁の**載りを封止材**にして初めて効く |

- 悲壮劇にしない。**作業と信頼**。  
- 逆鉾は**立てるだけでは載らない（効かない）**——言仁の載りが**ゴムパッキン**になって神気が止まる。  
- 史実名・蒙古等は**設定正本のみ**。本文NG。
