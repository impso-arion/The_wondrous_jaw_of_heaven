## 逆鉾を抜く・神気流入（霧島）★

> **人格神は本文に降ろさない。** 降りてくるのは**太陽神（日の神）系の神気の潮流**まで。  
> アマテラス本人・タカミムスビは出さない（高天原へ舞台を移さない）。  
> 「アマテラス」「日の巫女」等の**神話名は、土持ら平家方の誤読・美化**として使う（→ `../天/01_吾妻鏡_九州南部アンカー候補.md`）。

| 項目 | 内容 |
|------|------|
| **栓を抜く** | **天孫降臨の地＝霧島**の**天逆鉾**を抜く。神気の流入を止めていた栓が外れる |
| **神気の潮流** | 天から**汚染のように**流れ込む。空中ではなく、山の**水脈・霧・雨・潮の目**を伝う |
| **依代の人々** | 神気を浴びて、**祈らずとも**言仁と同じ**「載り＋人」**になれてしまう |
| **木花** | **太陽の依代体質**——神気に**最も強く反応する**核。殺さず**使う** |
| **言仁** | 生まれつき**正規キー（しるし）**の器。崇徳は新しい載りの**管理者キー**として要求する |
| **狙い** | 新しい「載り＋人」を量産し、**日向に新しい天孫族の国**を作る |

- 木花を殺すのではない——**使う**。第3部・言仁が見る「馬上の尊い姿」と接続。  
- 崇徳の「神気を**中へ入れる**（人に載せる）」＝言仁の「**外へ抜く**（栓で止める）」と**正反対**（`02_神器たるしるし.md`）。  
- 太陽神どまりにする理由 → `../天/01_吾妻鏡_九州南部アンカー候補.md`（敵側の神話誤読・土持の所有欲）。

## 言仁を追い詰める（精神的ラスボス）

> 言仁側の描写 → `01_言仁.md`

| 崇徳の戦略 | 効果 |
|------------|------|
| **緒戦有利** | 崇徳の**思惑通り**——言仁は**疲弊** |
| **抱え込み利用** | 「私がいなければ解決できない」焦りを**増幅** |
| **王者の孤独** | 「俺が生まれてきたから」「力がなければみんなと一緒に」——**自責**を深める |
| **木花への甘え→拒否** | 言仁の**人間的弱さ**を突く（崇徳は**同情しない**——**隙**として使う） |

- 崇徳は**休めと言う竹崎**への喧嘩には**直接出ず**、潮と**間接**で削る。  
- **精神的に追い詰められる**＝ラスボスらしさ。台詞は**少なく圧で**。

## ラストバトルへの道（敗北の形）

### 1. 挑発

- 「**崇徳、お前の国は、ここにあるぞ！　取りに来い！**」（言仁側の叫び——`03_言仁`）  
- 国＝山・載り・標的——**誘い込み**。

### 2. 載りを外した言仁

- 言仁が**自分の載りを外し**、それを**新しい逆鉾の封止材**にする——崇徳の**神気流入計画**が**空振り**する段階へ。  
- 崇徳「**ニニギ**」要求（正規キーを明け渡せ）は、**栓を打たれたあと**の焦燥としても可。

### 3. 三人による終止

| 役 | 行動 |
|----|------|
| **言仁** | 唱える。**自分の載りを抜き**、逆鉾に**封じる** |
| **竹崎** | 押さえる。人間の手。**新しい逆鉾を立てる** |
| **木花** | **載りの上流**（穴）を身体で押さえる。依代解除。前衛 |
| **共同** | 残った**分霊アイテム**（形代）を**破壊**——神気流入・再降臨経路を断つ |
| **天の逆鉾** | **新しく立てる**。**刃が天を向く**——栓。**言仁の載りを封止材**にして初めて神気が止まる |

- 逆鉾＋封止材で**神気の潮流が止まり**、崇徳の**載る先がなくなる**＝**パージ**（悲壮な別れ・改心**不要**）。  
- ログ一行程度：「目的は日本ではなかった」——**同情NG**のまま（世界観正本）。
