## 言仁を追い詰める（精神的ラスボス）

> 言仁側の描写 → `01_言仁.md`

| 崇徳の戦略 | 効果 |
|------------|------|
| **緒戦有利** | 崇徳の**思惑通り**——言仁は**疲弊** |
| **抱え込み利用** | 「私がいなければ解決できない」焦りを**増幅** |
| **王者の孤独** | 「俺が生まれてきたから」「力がなければみんなと一緒に」——**自責**を深める |
| **木花への甘え→拒否** | 言仁の**人間的弱さ**を突く（崇徳は**同情しない**——**隙**として使う） |

- 崇徳は**休めと言う竹崎**への喧嘩には**直接出ず**、潮と**間接**で削る。  
- **精神的に追い詰められる**＝ラスボスらしさ。台詞は**少なく圧で**。

## ラストバトルへの道（敗北の形）

### 1. 挑発

- 「**崇徳、お前の国は、ここにあるぞ！　取りに来い！**」（言仁側の叫び——`03_言仁`）  
- 国＝山・載り・標的——**誘い込み**。

### 2. バッファを外した言仁

- 言仁が**しるし摘出**／人間の身体——崇徳の**載せ替え計画**が**空振り**する段階へ。  
- 崇徳「**ニニギ**」要求は、**抜かれたあと**の焦燥としても可。

### 3. 三人による終止

| 役 | 行動 |
|----|------|
| **言仁** | 唱える。抜く。返天 |
| **竹崎** | 押さえる。人間の手 |
| **木花** | 依代解除。前衛 |
| **共同** | 残った**分霊アイテム**（形代）を**破壊**——再降臨経路を断つ |
| **天の逆鉾** | 立てる。**刃が天を向く**——降臨を**刺し止める** |

- **返天**で崇徳の**載る先がなくなる**＝**パージ**（悲壮な別れ・改心**不要**）。  
- ログ一行程度：「目的は日本ではなかった」——**同情NG**のまま（世界観正本）。
