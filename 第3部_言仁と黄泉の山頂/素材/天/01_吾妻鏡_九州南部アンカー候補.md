## 位置づけ

第3部のファンタジー部分を、鎌倉幕府による九州南部支配の進行と重ねるための素材。

歴史上は、幕府が日向・大隅方面の残党・悪党を追捕し、南九州支配を固めたことになっている。  
物語上は、その裏で、言仁たちが**古墳に眠る超常の配線**と、崇徳の再降臨計画を断っていた、という二重構造にできる。

## 1. 肥後・宇都宮：武藤資頼という管理者候補

### 史実アンカー候補

| 項目 | 内容 |
|------|------|
| **人物** | **武藤資頼**（むとうすけより） |
| **時期** | 文治5年（1189年）正月条あたりの吾妻鏡メモとして要検証 |
| **出自・経歴** | かつて平知盛に仕えた囚人だったが、有職故実に詳しかったため頼朝に罪を許され、御家人に取り立てられたとされる |
| **後の位置** | 九州の**大宰少弐（少弐氏）**へつながる人物として扱える |
| **接続候補** | 肥後国の平家没官領、旧原田種直領、地頭職、豊前宇都宮氏初代・信房との関係 |

### 第3部での使いどころ

第1〜2話の「起」で、**幕府・宇都宮・肥後**の三層すべてに足を突っ込んでいる、不気味な管理者として配置できる。

| 層 | 武藤資頼の機能 |
|----|----------------|
| **幕府** | 頼朝のシステムに取り込まれた、元・平家側の人間 |
| **肥後** | 平家没官領や旧原田領の管理者として、竹崎の地元に圧をかける |
| **宇都宮** | 宇都宮側が何を隠しているか、気配を察知できる監視者 |

### 物語上の役割案

- 竹崎たちに「日向の残党・悪党追捕」を命じる、または条件として突きつける。
- 原田種直の残した領・平家没官領・宇佐の網を知っている。
- 味方ではないが、単純な悪役でもない。**幕府システム側の冷たい人間**。
- 竹崎から見ると「平家側だったくせに、いまは幕府の帳簿の顔をしている」人物。

## 2. 日向東征：悪党・残党追捕という軍事名目

### 史実アンカー候補

| 項目 | 内容 |
|------|------|
| **時期** | 建久年間（1190年代）候補 |
| **地域** | 日向国・大隅国の国境付近、山地 |
| **対象** | 幕府の統制に従わない**悪党**・**平家残党** |
| **幕府の動き** | 鎌倉から断続的に**追捕**の命令が出る、という形で使える |
| **在地勢力** | 税所氏、のちの島津氏につながる在地勢力など、独自勢力が割拠 |

### 第3部での使いどころ

第3〜6話の「承」で、竹崎たちが日向へ向かう大義名分にできる。

表向きは、幕府または肥後側から下された**日向の残党・悪党退治**。  
裏では、崇徳が日向の山・古墳・怨みを使い、古代の配線にアクセスしている。

| 表 | 裏 |
|----|----|
| 幕府の命令による悪党狩り | 古墳に眠る国津・寝る神系の潮流を断つ |
| 日向山地への軍事東征 | 崇徳の古墳ソケットを潰す |
| 残党追捕 | 再降臨の配線追跡 |

### 使える短句

- 「日向の山に、まだ従わぬ者がいる」
- 「悪党狩りだと、帳簿には書ける」
- 「人の残党を追っていたら、墓が潮を吸っていた」
- 「幕府の追捕と、崇徳の東征が、同じ山道で重なった」

## 3. 高千穂・逆鉾：惟宗忠久の南九州下向

### 史実アンカー候補

| 項目 | 内容 |
|------|------|
| **人物** | **惟宗忠久**（これむねただひさ）＝島津氏初代 |
| **時期** | 建久8年ごろ候補 |
| **立場** | 頼朝の寵臣。南九州の広大な土地、**島津荘**の守護・惣地頭として入る |
| **地域** | 日向・薩摩・大隅方面 |
| **物語での機能** | 幕府の最新兵力が、南九州完全支配のため山麓まで迫る |

### 第3部での使いどころ

第7〜10話の「転・結」で、山頂決戦の**物理的な時間制限**にできる。

言仁たちが目指す高千穂・霧島の山頂へ、崇徳の儀式だけでなく、鎌倉幕府の軍勢も近づいてくる。  
幕府軍が山頂を完全包囲する前に、**天孫返天**と**しるし摘出**を終えなければならない。

| 要素 | 使い方 |
|------|--------|
| **惟宗忠久の軍勢** | 南九州完全支配のために山麓へ迫る、最新の幕府兵力 |
| **松明** | 山を埋め尽くすタイムリミットの絵 |
| **逆鉾** | 幕府軍が山頂に着いたとき、残っているのはただの逆鉾だけ |
| **三人の足跡** | 超常を失い、人として去った痕跡。歴史から消える |

### 山頂決戦の絵

- 幕府軍の松明が山の下を埋める。
- 山頂では、崇徳の再降臨儀式と返天がぶつかる。
- 竹崎が言仁を押さえ、しるしを抜く。
- 木花の依代が外れ、超常の回線が閉じる。
- 天の逆鉾が立つ。
- 幕府軍が到達したときには、神も怪異も言仁もいない。

## 敵対者候補：平家方残党・忠誠の歪み

> こちらも**要検証素材**。  
> 武藤資頼・惟宗忠久を「幕府システム側の圧」として置くなら、その反対側に、**平家への忠誠を捨てられず、古墳・天孫・怨霊へ縋る者**を置ける。  
> 言仁にとっての地獄は、かつて自分を支えた者たちを、鎌倉幕府の駒として**悪党・反逆者として切り捨てねばならない**こと。

### 1. 本命：土持信栄（つちもち のぶひで）

| 項目 | 内容 |
|------|------|
| **位置づけ** | 日向国・高千穂周辺に強い勢力を持つ土持氏の当主候補 |
| **陣営** | 平家方。源平合戦で平家を支えた九州の在地勢力として扱える |
| **物語上の役** | **日向・高千穂の本来の支配者**。古墳・高千穂神話・国津の潮を使い、源氏を呪う黒幕候補 |
| **崇徳との接続** | 崇徳本人の部下というより、**平家再興の怨み**を崇徳に利用される、または自ら利用する |
| **言仁への刺さり** | かつて安徳を支えた忠臣が、言仁の前に跪き、狂信的な忠誠を捧げる |

#### 第3部での使い方

土持信栄は、第7〜9話の山頂前後で最も効く。  
彼は言仁に向かって「陛下」と呼び、古代の神々の力で鎌倉を滅ぼそうとする。

| 表 | 裏 |
|----|----|
| 平家への忠義 | 言仁を**安徳**へ戻そうとする鎖 |
| 高千穂の支配者 | 山頂の罠の**人間側の顔** |
| 源氏への呪い | 崇徳の再降臨計画と噛み合う怨み |
| 日向の悪党 | 幕府の帳簿上は討伐対象 |

#### 言仁の葛藤

- 土持氏は、かつて都落ちした言仁（安徳）を九州で支えた恩人筋として扱える。
- 土持信栄は「**陛下、今こそ古代の神々の力をもって、鎌倉を滅ぼしましょう**」と忠誠を捧げる。
- しかし言仁は、すでに**天孫返天＝超常の消失**を決意している。
- そのため、鎌倉側の軍勢（武藤・惟宗・北条）を日向に引き入れ、誰よりも自分を愛し、忠誠を誓う者を、**幕府の反逆者として討つ**地獄を味わう。

#### 使える短句

- 「陛下。高千穂は、まだあなたの山です」
- 「源氏を呪うなら、墓はいくらでも眠っております」
- 「忠義が、いちばん重い鎖になることがある」
- 「俺は、俺を助けた者を、悪党として討ちに来た」

| 項目 | 内容 |
|------|------|
| **位置づけ** | 原田種直の子ら。筑前の所領を失い、九州各地に潜伏する残党候補 |
| **陣営** | 平家方の没落一族。父の処遇・所領没収への恨み |
| **物語上の役** | 土持氏と結託し、日向の悪党蜂起に加わる。怨霊・古墳・平家納経系の残響を起動する触媒 |
| **言仁への刺さり** | 「種直のおかげで生き延びた」という負い目がある相手 |

#### 第3部での使い方

第3〜6話の古墳東征で、悪党・残党側の顔として使える。  
彼らは「父は源氏の奴隷にされた」「領地を奪われた」と信じ、復讐に狂う。

| 表 | 裏 |
|----|----|
| 原田の残党 | 第1部の敗北と生存の後始末 |
| 源氏への復讐 | 崇徳院の怨載りと接続しやすい |

#### 言仁の葛藤

- 言仁たちは、原田種直の判断によって生き延びた。
- しかし息子たちは、言仁が生きている事実を知らず、ただ源氏への復讐に狂っている。
- 言仁は恩人の息子たちを救えず、鎌倉の公式な命令として切り捨てねばならない。

#### 使える短句

- 「お前たちの父に、俺は生かされた」
- 「知らない恩ほど、返しようがない」
- 「原田の名が、また墓の前に立っていた」

### 3. 那須宗久（なす むねひさ）

| 項目 | 内容 |
|------|------|
| **位置づけ** | 日向国椎葉村の平家落人伝説に接続する人物候補 |
| **陣営** | もとは幕府側の追討使。現地の平家残党に同情し、庇う側へ回る |
| **物語上の役** | 源氏側でありながら平家のために戦う、立場反転の敵 |
| **言仁への刺さり** | 言仁自身は、平家の王でありながら鎌倉のシステムを利用している |

#### 第3部での使い方

第5〜7話あたりの、言仁の孤独・喧嘩・木花の孤独と重ねると効く。  
那須宗久は「平家のために、源氏を捨てる」男。  
言仁は「人として生きるために、源氏の仕組みを使って超常を消す」男。

| 那須宗久 | 言仁 |
|----------|------|
| 源氏側なのに、平家のために源氏を捨てる | 平家の王なのに、源氏のシステムを使う |
| 情で残党を庇う | 冷徹に超常を終わらせる |
| 現地の人間になる | 歴史から消え、人として去る |

#### 言仁の葛藤

- 那須宗久は「平家のために命を捨てる」と言って敵になる。
- 言仁は、自分たちのために源氏を裏切ってくれた男を、鎌倉側の立場で討たねばならない。
- 「誰が平家を裏切ったのか」が反転する。

#### 使える短句

- 「源氏を捨てた源氏が、平家の王の前に立った」
- 「お前のほうが、よほど平家らしい」
- 「俺は、誰のために源氏の列にいる」

## 敵対者候補の比較

| 候補 | 一番効く回 | 効き方 | リスク |
|------|------------|--------|--------|
| **土持信栄** | 7〜9 | 高千穂・山頂・古墳・平家忠誠が一本化する。言仁を**安徳へ戻す鎖**になる | 情報量が大きい。崇徳との役割分担を明確にする必要 |
| **原田の息子たち** | 3〜6 | 第1部の負債を回収できる。悪党追捕の顔として使いやすい | 原田種直本人の扱いと混線しやすい |
| **那須宗久** | 5〜7 | 立場反転の悲劇が強い。言仁の孤独・喧嘩に刺さる | 伝説色が強く、採用すると別筋のドラマが増える |

### 現時点の採用感

**本命は土持信栄。**  
理由は、第3部の既存プロットである**日向・高千穂／古墳東征／山頂の罠／天孫返天**と、最も直結するため。

ただし、土持信栄を「崇徳の代わりのラスボス」にしない。  
崇徳はあくまで**超常システム側の主催者**。土持信栄は、言仁を安徳へ戻そうとする**人間側の忠誠の歪み**として置くのがよい。

## 敵側女性キャラクター候補

> 男性武将・幕府軍だけだと画面が泥臭くなりすぎる。  
> 第3部の「古墳東征」「山頂の罠」「天孫（ニニギ）」に、**神話・呪術・怨念・木花との鏡像**を足すための女性キャラ候補。  
> こちらも史実／伝説／創作混合の**要検証・要選別素材**。

### 1. 鶴富姫（つるとみひめ）の裏の顔

| 項目 | 内容 |
|------|------|
| **由来** | 日向国椎葉村の平家落人伝説のヒロイン候補 |
| **伝説上の姿** | 那須宗久と恋に落ち、子を宿す。宗久が鎌倉へ戻った後も現地に残り、一族を守る強い女性 |
| **第3部での歪み** | 日向に隠れ潜む平家残党の**若き巫女**、または女マフィア的な残党の顔 |
| **呪術機能** | 日向の古墳に眠る古代の配線を理解し、平家の怨念を天孫（ニニギ）の力と結びつける |
| **言仁への刺さり** | 「安徳天皇が生きていた」と知り、狂喜して言仁を**陛下**へ戻そうとする |

#### 使いどころ

鶴富姫型は、言仁の罪悪感と「平家のために生きるべき」という誘惑に効く。

- 「陛下、この日向の神話の力で、今度こそ鎌倉を呪い殺しましょう」
- 言仁にとって、彼女は**自分と平家のために、人生のすべてを呪いに捧げてくれた同世代の少女**。
- だからこそ、鎌倉のシステム側に立って切り捨てるのが重い。
- 那須宗久を採用する場合、その悲恋・源氏側の裏切りと接続しやすい。

#### 注意

- 鶴富姫を使うと、椎葉・落人伝説のドラマが強くなる。第3部の主軸（返天・摘出）を奪わないこと。
- 悲恋を厚くしすぎると、木花の孤独・言仁の摘出が薄まる。

### 2. 土持妙（つちもち たえ／仮名）— 土持一族の神和ぎの娘

| 項目 | 内容 |
|------|------|
| **由来** | 土持氏まわりからの創作キャラ。土持当主の妹・娘など |
| **位置づけ** | 高千穂の神々を祀る一族の娘。**山頂の罠そのもの**の人間側の顔 |
| **役割** | 神話システムをハックする天才呪術少女。承の疲弊、転の山頂罠に関わる |
| **自負** | ニニギの妻・木花開耶姫の血を引く、またはその依代であると信じている |
| **木花との関係** | 木花（このは）の敵側鏡像。「偽物の木花が、陛下の隣にいるな」と敵意を向ける |

#### 使いどころ

土持妙型は、**木花の騎馬・孤独・ジサイばなれ**に強く刺さる。  
第4〜7話で、木花に対するライバル呪術少女として出すと、画面が華やかになりつつ、精神的にドロドロする。

| 木花 | 土持妙 |
|------|--------|
| 家族になりたい。真ん中にいるのに孤独 | 血統・神話・役目で自分を支える |
| 騎馬の前衛。人間として立つ | 神和ぎ・古墳・高千穂の呪術 |
| 依代を外され、ただの少女へ | 依代であることを誇り、捨てられない |
| 言仁を人として見る | 言仁を**陛下／ニニギ**へ戻そうとする |

#### 使える短句

- 「偽物の木花が、陛下の隣にいる」
- 「あなたは家族になりたいだけ。私は、神に選ばれた」
- 「馬で来るなら、山で落としてあげる」
- 「高千穂の名を、そんなふうに軽く呼ばないで」

#### 注意

- かなりフィットするが、名前・血筋・神話設定を盛りすぎると講義になる。
- 崇徳の「ニニギ」要求と噛み合わせるなら、土持妙は**人間側の実行者／巫女**に留める。
- 木花との対立は恋愛三角ではなく、**依代観・家族観・人間として立つこと**の対立にする。

### 3. 平盛子（たいらの もりこ）または娘世代

| 項目 | 内容 |
|------|------|
| **由来** | 平家一門の女性たち、地方へ配された高貴な姫君という発想 |
| **位置づけ** | 平家の全盛を知る、誇り高い姫君。日向の辺境で没落しても目だけは爛々としている |
| **役割** | 言仁が鎌倉のシステムを使って生き延びることを、**平家の恥**として責める |
| **呪術機能** | 言仁を「本物の安徳天皇の怨霊」へ再起動しようとする |
| **刺さり方** | 言仁の「人として生きる」選択を、平家の誇り・清盛の呪いで否定する |

#### 使いどころ

平盛子型は、言仁の内面へ最も直接刺さる。  
第8〜9話の返天前に、言仁を**安徳の怨霊**へ戻そうとする誘惑・断罪として使える。

- 「鎌倉の印を借りて生きるなど、平家の恥です」
- 「陛下ではなく、人として生きる？　それは清盛公への裏切りです」
- 「あなたが死なぬなら、せめて怨霊として立ちなさい」

#### 注意

- 強いが、崇徳院と役割が被りやすい。  
  崇徳＝超常システムの主催者、盛子型＝**平家の誇りによる心理的断罪**に分ける。
- 清盛・平家全盛の説明を長くしない。言仁への一撃として短く。

## 女性敵候補の比較

| 候補 | 一番効く回 | 効き方 | リスク |
|------|------------|--------|--------|
| **鶴富姫型** | 5〜7 | 平家残党の若き巫女。言仁を**陛下**へ戻す健気な狂信 | 椎葉・那須宗久筋が膨らみすぎる |
| **土持妙型** | 4〜7 | 木花の鏡像。騎馬・呪術・高千穂・山頂罠に直結 | 設定を盛ると講義になる。崇徳の役割を奪わない |
| **平盛子型** | 8〜9 | 平家の誇りで言仁を断罪。返天前の誘惑に強い | 崇徳と心理攻撃が被る。重くなりすぎる |

### 現時点の採用感

**第3部の既存プロットに一番きれいに入るのは、土持妙型。**  
理由は、木花の第3部要素（騎馬・孤独・依代解除）と、高千穂・山頂の罠を同時に強化できるため。

ただし、言仁の葛藤を最大化するなら、土持妙に**鶴富姫型の「陛下」狂信**を少し混ぜるのがよい。  
つまり、単独の新キャラ案としては：

| 仮採用案 | 内容 |
|----------|------|
| **土持妙** | 土持氏の神和ぎの娘。高千穂の巫女。木花の鏡像 |
| **鶴富姫要素** | 若き平家残党の姫／巫女として、言仁を「陛下」と呼ぶ |
| **役割** | 木花には「偽物の木花」と敵意。言仁には「安徳へ戻れ」と忠誠 |
| **終盤** | 山頂の罠の人間側の実行者。崇徳の再降臨に利用される、または自ら加担する |

この場合、土持信栄は当主・政治軍事の顔、土持妙は神和ぎ・山頂呪術の顔として分担できる。

## 採用時の注意

- **要検証素材**として扱う。史実の厳密性が必要になったら、吾妻鏡該当条・自治体史・論文で確認する。
- 本編の主役は史実人物ではなく、**竹崎・言仁・木花・崇徳**。
- 武藤資頼・惟宗忠久を出す場合も、長く内面化しない。**システム側の圧**として短く効かせる。
- 土持信栄・原田の息子たち・那須宗久を出す場合も、崇徳の役割を奪わせない。彼らは**人間側の忠誠・怨み・反転**。
- 女性敵を出す場合は、木花の成長線（騎馬・孤独・ただの少女へ）を強めるために使う。恋愛三角・嫉妬だけにしない。
- 史実名を増やしすぎると第3部の焦点が散る。採用は各回1要素まで。
- 第3部のテーマは、南九州平定ではなく、**超常システムの停止と人間への明け渡し**。
