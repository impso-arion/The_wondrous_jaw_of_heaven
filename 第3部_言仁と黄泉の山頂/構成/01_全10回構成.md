## 形式

| 項目 | 内容 |
|------|------|
| **話数** | **10回**（約4〜4.5万字） |
| **1回** | 約4000字。起・承・**転（バトル800〜1200字）**・結＋**次回への崖** |
| **毎回** | **バトルあり**（各回詳細・下記）。ただし司馬は**短い一段**まで。軍勢でも地の文の長文化は避ける |

| **言仁の名** | 公式・軍では**宇都宮**。竹崎の前では**言仁**。決戦後**剥落** |
| **やらない** | 言仁の**殉教・散華**。安徳の**悲劇的死**で締める。言仁が**大軍を直接**打つ（テーマ崩れ） |

## 転部（7〜10）の原則

| 項目 | 内容 |
|------|------|
| **言仁が大軍を直接打たない** | 外寇・大軍を言仁の魔術で薙ぎ払う展開は**テーマ崩れ**。決戦の主役は**人間の軍勢＋三人の作業** |
| **崇徳は圧で出す** | 長文説得・哀愁・改心**NG**。「**計画通り**」「**山へ登れ**」型の短い圧。内面独白は崇徳視点も基本禁止 |

| **死で締めない** | 言仁は**生きる**。安徳の悲劇的死で終わらせない。返天後は**回線が閉じた**（神託が来ない）静けさ |
