## ★ 呪文生成レギュレーション（掛詞・ダブルミーニング）

**物部流呪文の芯は、深層で「その場でやりたいこと」が立っていること。**  
表層は祝詞・歌・本の一節に**似せてよい**——むしろ、言仁は既存の文を**言いなおす**のが得意。

### 優先順位

1. **効果**（反魂／尖刺／…）を先に決める  
2. 効果を表す語を、古語・音便・助詞で**二重に読める**形に潰す  
3. 表層：布瑠の言・数え歌・**既存の祝詞**に紛れる響き  
4. 深層：**やりたいことそのもの**を言っている  

### 言仁の「言いなおし」（ファンタジー設計）

**執筆上の意図**：新呪文を毎回ゼロから発明しなくてよい。  
言仁（と作者）は、手元のテキストから**容易に**物部流を組み立てられる。

| 場面 | やること | 例 |
|------|----------|-----|
| 第1部・海 | 育ての親や船から聞いた歌・断片を言いなおす | ふるべ系 → 尖刺 |
| 第2部・学園 | **本を読んでいる**最中に、祝詞の一節を物部流に差し替える | 厳島の祝詞 → 防御用の掛詞 |
| 第3部以降 | 秘伝・反魂・天孫返天へ拡張 | 完全形ふるべ |

**言いなおしの手順**

1. 既存の祝詞・歌・文献の**音とリズム**を借りる（表層）  
2. 一語ずつ、または句ごとに**深層＝効果**が立つ掛詞へずらす  
3. 短く切る。長い整文のまま唱えない  

※ 天孫系が「格式どおり唱える」なら、言仁は「**同じ本を、違う意味で唱える**」。  
竹崎から見れば、どちらも難しい早口だが、**効き方だけが違う**。

### 天孫祝詞との差

| | 天孫系（神社・宗像） | 物部流（言仁） |
|--|----------------------|----------------|
| 目的 | 格式・権威・正規アクセス | **効果の掛詞** |
| 本・祝詞 | 正しく唱える | **言いなおして**唱える |
| 聞こえ方 | 「きちんとした祝詞」 | 似ているが**中身が違う** |
| 竹崎の感想 | 「祈祷」 | 「数え歌」「変な早口」 |

### 掛詞の作り方（手順）

| 段 | やること | 例（反魂） |
|----|----------|------------|
| 1 | 効果を一文で言う | 「古い魂を、揺らして降ろす」 |
| 2 | 音節を二重読みに分解 | ふる＝古／振る、ゆら＝揺ら、とふる＝と降る |
| 3 | 布瑠の言の**型**、または既存祝詞の**型**に載せる | …ふるべ　ゆらゆらと　ふるべ |
| 4 | 本文では講義しない | 刺した／魂が戻った、で十分 |

### やってはいけないこと

- 深層（効果）のない**引用だけ**の呪文  
- 一語一義で掛詞を潰す（例：とふるべ＝「締め」だけ、等）  
- 本文での呪文語源講義  

### 新呪文を足すときのチェック

- [ ] 深層：この呪文で**何をしたいか**が一文で言えるか  
- [ ] 表層：布瑠の言／祝詞／数え歌系の**音**に乗っているか  
- [ ] 各句に**二つ以上の読み**（または音の重なり）があるか  
- [ ] 言いなおしなら、**元の文との対応**が作者メモで説明できるか  
- [ ] 本文で呪文の語源講義をしていないか

## 身体（たたき台）

**原則**：酸素を多く取る・精神が安定する・体が整う。呪術用の「必殺ポーズ」一本足打法にしない。

### 基本（優先）

| 順 | 内容 | 目的 |
|----|------|------|
| 1 | 足裏で地面／甲板を感じて立つ | 安定 |
| 2 | **腹式呼吸**（鼻から吸い、長く吐く） | 酸素・落ち着き |
| 3 | 背筋を延ばす（胸が開く） | 肺・姿勢 |
| 4 | 外（海の音・風）のリズムに**吐息**を合わせる | 潮合（呼吸側の意味） |
| 5 | 合図：腹の底が冷える／指先が静かになる／息が一瞬止まる | 小指痺れも合図の一つとして可 |

### 例として残す型（必須ではない）

**肩を下げる型**（言仁がよく取る一例）

1. 首を伸ばし、肩をストンと下ろす  
2. 脇を締め、指先の感覚が変わる  
3. 小指がピリピリする  
4. 続けて布瑠の言を唱える  

※ 他の健康法と同列。**肩下げ＝物部流の本体ではない**。

## 言霊の根：布瑠の言（ふるのこと）

### 完全形（反魂・秘宝系）

```
ひふみよ　いむなや　ここのたり
ふるべ　ゆらゆらと　ふるべ
```

**一句訳（執筆用・深層）**：「**古き魂が、ゆらゆらと降るところ**」

| 句 | 表層（耳に馴染む） | 深層（効果＝反魂） |
|----|-------------------|-------------------|
| ひふみよ… | 数え上げ・布瑠の言の前段 | 世界を音の順でほどく |
| **ふるべ** | 振るべ、古（ふる）… | **古**き魂を扱う |
| **ゆらゆらと** | 揺ら揺らと | 魂を**ゆら**して**と**（接続） |
| **とふるべ** | と振るべ？ | **と降るべ**（揺れ落ちて還る） |

- **銛の先鋭化・第1部の戦闘には使わない**（反魂専用）。
- 第3部・天孫返天＋**しるし摘出**（尖刺の逆）→ `../../第3部_言仁と黄泉の山頂/設定/01_言仁の結末と神器たるしるし.md`

### 古代語ファンタジー（執筆用）

- 表層は祝詞・歌に似せ、深層は**当時の効果語**——というズレがある。
- 言仁は本や断片から**言いなおせる**。竹崎は「は？」。

## 銛強化（第1部・代表呪文たたき台）

**型名**：布瑠一之型・**尖刺（せんし）**

**身体**：基本呼吸 → 海の音に吐息を合わせる（潮合＝**呼吸**）

**効果（深層）**：海の揺らぎを止め、古い力を**尖って降ろし**、刺す。

**詠唱**

```
ひ　ふ　み
ふる、尖（とが）れ、降（ふ）れ
ゆら、止（と）め
と　刺（さ）せ
```

| 句 | 表層 | 深層（効果＝尖刺） |
|----|------|-------------------|
| ひふみ | 布瑠の言の入口 | 数え上げで刃をほどく |
| **ふる、尖れ、降れ** | 古…尖…降… | **古**き力を**尖**らせ**降**ろす |
| **ゆら、止め** | ふるべのゆら…を借りる | 海の**揺らぎを止める** |
| **と刺せ** | とふるべの型を借りる | **と**（接続）**刺せ** |

**竹崎の語り**：「数え歌みたいなもん唱えて、刺した」

※ 型名「尖刺」は設定メモ用。詠唱中に漢字「尖刺」を唱えない。
