## この世界の「神」とは

| 層 | 古事記（表の神学） | 本作の実体 |
|----|-------------------|------------|
| **天津** | 天孫降臨・国譲り・神武東征 | **譲渡の手続き**（天から国を受け取るプロトコル） |
| **国津** | 大国主を育てて国を譲らせる | **土地の結節**（古墳・石室・海のクジラ等） |
| **社稷** | 三種の神器＝王権の証 | **国そのものが入っている器**。神が**宿る素材** |

**神＝天にいる人格、ではない。**  
**国（社稷）に「載っている」状態**が、この世界の超常の正体。  
→ 用語要約 **`04_載りとは.md`**

- 朝廷・神社・依代は**中継**（ルーター・アクセスキー・インターフェース）  
- 天皇の正当性は天津①の**公式**として残る  
- しかし**本物の国**は③の神器に載っていた

## 壇ノ浦以降（fiction の分岐）

```
本来：天 → 譲渡 → 神器（社稷）→ 依代・社・王
現在：天 → 譲渡 → 【三種沈没】→ 依代だけが無理やり中継
```

| 項目 | 内容 |
|------|------|
| **沈没** | 壇ノ浦で失ったのは「証明書」ではなく**国の本体** |
| **草薙** | 紛失（史実どおり）。剣そのものはもうない |
| **残存** | 天皇家の血に**神器たるしるし**——載せ替え可能な**署名・根っこ**だけが残る |
| **依代** | 本体なしの**中継**。力を失いかけている |

### 神器たるしるし

- 三種の**代わり**ではない。**載るためのハッシュ**（管理者キー）  
- 安徳＝言仁の血に**埋め込まれている**  
- 崇徳の「**体が国**／受肉」＝このしるしを**自分側に書き込む**野望  
- 言仁の拒否＝**神器＝自分ではない**（第2部 `02_神器の鳥居.md`）

## 平家物語との距離（執筆方針）

> **「安徳が結局死ぬんか」で終わらせない。**

| 平家物語の読み | 本作 |
|----------------|------|
| 安徳天皇の悲剧的死亡 | **回避**（fiction） |
| 幼帝＝国の象徴 | 幼帝の血に**しるし**が載る |
| 神器沈没後の空虚 | 言仁が**自らしるしを抜き**、**人として生きる** |

史実の安徳の死を**否定する史実小説**ではない。  
**if：載りを自分で外した**という別結末。

## 神との対話（三部共通）

> **メタ（1部メルヘン／2部ファンタジー）ではなく、設定上の「回線の種類」。**  
> 三部とも神（結節）との対話**してよい**。ただし**同じ神ではない**。

### 四種類

| 種類 | 例 | 対話の質 |
|------|-----|----------|
| **天津・中継** | 沖ノ島（宗像）、厳島、宇佐 | 警告・神託・格式。**潮経由の公式回線** |
| **国津・結節** | 古墳・石室・石人、出雲系 | 夢・静寂・偶発。ヤマト祝詞が**通じにくい** |
| **海の隔離** | 死者のクジラ、母 | 愛・執念。**神というより残響**（第1部で使い切り） |
| **人間側** | 木花・ジサイ | **神ではない**（外部神頼みを使わない）——対話の対照 |

- **対話できる**＝結節に**載っている／回線が開いている**状態  
- 厳島＝学園OSの**表の回線**／古墳＝**国津の古いソケット**／沖ノ島＝**海の正規ルータ**  
- 第3部・古墳戦の「どの神様か」は**国津・寝る神・石人の主**など（ヤマト天津と別系統）

### 執筆ルール（対話）

- 講義しない。竹崎語り：**返事・機嫌・潮・聞こえにくい**  
- 木花・ジサイの祈祷は**天津中継と別物**（温かい生気 vs 冷たい神光）

## 天孫返天後——「神が死ぬ」のではない

**対話できる回線が閉じる。** 載せ替え契約終了＝**会話のポートがない**。

```
返天前：潮・社・依代・古墳 → 声が返る（条件付き）
返天後：同じ場所へ行っても → 石・海・風。返事が来ない
```

| 対象 | 返天後 |
|------|--------|
| **崇徳** | パージ。会えなくてよい |
| **宗像・厳島** | 神格の**死**ではなく、**神託が届かない**（信仰・社は残っても本作の超常回線は閉じる） |
| **古墳** | **ただの遺構**。もともと「話す」より眠っている |
| **木花** | 依代アンマウント。**普通の人間**（神と会えないのではなく載りが外れた） |
| **出雲** | 国譲り旧OS——**静まる**（一行触れてよい） |

- 蒙古・神風なしと同型：**奇跡の回線がない**  
- 竹崎：「**前みたいに返事が来ねえ**」「**潮に乗らねえ**」——**機能停止**として触れる  
- 返天後の喪失を**全巻の主題**にしない。第3部結 or エピローグ**1シーン／1行**で足りる

## 第1部・母の言葉（重ね読み）

> 「平家の栄華も天子の位も、陸の人間が作った虚しい幻。……**お前は、お前自身の国を、お前の足で探しなさい**」

第1部では**幽霊船から陸へ**の場所の指示。全文で**正解は一つに解かない**。

| 読み | 効くタイミング |
|------|----------------|
| **A** | 天子・平家の「国」は幻。**自分の足**で生きろ（母の本意） |
| **B** | 陸＝現実＝組織の世界へ（第1部末・宇都宮恭順） |
| **C** | **載り（しるし）**を外し、**人として**国を持て（第3部） |

- 第2部の宇都宮は一見 **B と矛盾**——読者の「**どっち？**」は**意図どおり**  
- 第3部で **A と C が重なって回収**。B は「一度通ったが、最終的に外した」  
- 本文で「正解はこっち」と**解かない**。竹崎のツッコミ程度でよい  
- 詳細 → `../../第1部_言仁と死者のクジラ/素材/地/02_海の隔離領域と死者のクジラ.md`

## 執筆ルール

- 「神器たるしるし」「社稷」「プロトコル」を**講義しない**  
- 竹崎の理解度で：**糸・結び目・潮の目みたいな光**程度  
- 第1部「死者のクジラ」は**使い切り**。摘出道具は**小片のみ**残る（→ 第3部設定）  

### 第1部・摘出道具（受け渡し）

| 道具 | いつ・誰から | 第3部での役割 |
|------|--------------|---------------|
| **解結の欠片**（沈んだ国の骨片） | 第8話・**母**が言仁の懐へ | しるし摘出の**抜き工具**（尖刺の逆） |
| **鯨油**（小） | 第1部・解体・荷物 | 何を外すか**見える**灯・鏡 |

- 母の台詞：「結び目」「のち、**自分**を、解く」——**載るな**（A）と**外せ**（C）の重ね読み  
- 本文：竹崎は**糸・結び目・小さい骨**程度。神器・摘出と**講義しない**
