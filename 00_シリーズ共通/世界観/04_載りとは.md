## 一言

**国（社稷）という「もの」に、誰か／何かが接続されている状態。**

本作の超常の正体は、天にいる神人格でも、偉い人間だけでもない。  
**載っている**＝祝詞が効く・神託が返る・依代が通る——**接続状態**。

## 比喩（作者用・本文NG）

| 言葉 | 本作での意味 |
|------|--------------|
| **国（社稷）** | 本体。もともと**三種の神器**に入っていた |
| **載り** | その本体に**接続されている**状態 |
| **依代・社・鳥居** | **中継**（ルーター）。本体は持たない |
| **神器たるしるし** | 載るための**署名・管理者キー**（壇ノ浦後、血に残る） |
| **祝詞** | 接続の**パスワード**（短い掛詞） |
| **潮** | **配線**。魔力は海水を伝う |

### 壇ノ浦以降

```
本来：天 → 譲渡 → 神器（国本体）→ 王・依代・社
現在：天 → 譲渡 → 【三種沈没】→ 依代だけが無理やり中継
                                    └ 血に「しるし」だけ残る
```

**載り＝壊れかけの契約。** まだ繋がっているが、本来の器はもうない。

## キャラ別——載りへの態度

| 誰 | 態度 | キーワード |
|----|------|------------|
| **崇徳** | 奪われた。取り返したい | 体が国／載せ替え／怨載り |
| **言仁（安徳）** | 血にしるしがあるが**自分＝国ではない** | 侮るな／しるし摘出 |
| **木花** | **依代**（無自覚） | 返天後＝普通の人間（アンマウント） |
| **母** | 載りそのものを疑う | お前自身の国を、お前の足で |
| **宇都宮** | 載りの**殻・看板**（第1部末） | 一度は「組織の姓」として載る |
| **定信・ジサイ** | 載り**しない**（人の命側） | 木花・祈祷の対照 |
| **竹崎** | 載りを**理論しない**。言仁の**肩・機嫌**で通訳 | 第3部：外す決断を**行動で支える** → `../登場人物/05_竹崎季長_第3部.md` |

### 方向の対立（核心）

| | 崇徳 | 言仁 |
|--|------|------|
| 方向 | **中へ入れる**（受肉・再起動） | **外へ抜く**（摘出・契約終了） |
| 国 | **恨む**＝奪還 | **拒否**＝自分ではない |

## 母の言葉との重ね読み

> 「**お前は、お前自身の国を、お前の足で探しなさい**」

| 読み | 載りとの関係 |
|------|--------------|
| **A** | 天子・平家の「国」は幻。**載りを名乗るな** |
| **B** | 陸＝組織の世界へ（宇都宮恭順）——**一度は載る** |
| **C** | **しるしを外し**、人として生きる（第3部） |
| **ラスト（物語の芯）** | **自分の中のバッファを外す**＝不思議パワーのない、**みんなと同じ人間の身体**（領地・神の王ではない） |

- 正解は**一つに解かない**（`02_神器たるしるし.md`）  
- 第3部で **A と C が重なる**。B は「一度通ったが、最終的に外した」  
- **竹崎**は言仁のラストに**多く影響しうる**（聞く・動かす・前衛）——`05_竹崎季長_第3部.md`

### 返天後

**神が死んだのではない。** **載りの契約が終わり、返事が来なくなる。**

- 崇徳：パージ（載る先なし）  
- 木花：依代アンマウント  
- 厳島・宗像：社は残っても**本作の超常回線は閉じる**  
- 竹崎：「**前みたいに返事が来ねえ**」「**潮に乗らねえ**」

## 本文でどう書くか

### NG

- 社稷・プロトコル・載せ替え契約の**講義**  
- 「載り」の**定義説明**をキャラがする  
- ハッキング用語の直書き（→ 竹崎の理解度で曲げる）

### OK（例）

| 誰 | 台詞・描写 |
|----|------------|
| **崇徳** | 「**体が国**」「**載せ替え**を」（短い・余） |
| **言仁** | 「……**侮るな**」（載せ替え拒否） |
| **竹崎** | 「あいつの肩、重い」「**抜く**、ってことか」（粗い言葉） |
| **竹崎** | 「**前みたいに返事が来ねえ**」（返天後） |
| **描写** | 糸・結び目・潮の目・載ってる／外れた |

読者の**崇徳怨霊・幼帝**前理解があれば、用語を一度も説明しなくても、**方向の逆**（崇徳 vs 言仁）で通じる。
