## 一言

**国（社稷）という「もの」に、誰か／何かが接続されている状態。**

本作の超常の正体は、天にいる神人格でも、偉い人間だけでもない。  
**載っている**＝祝詞が効く・神託が返る・依代が通る——**接続状態**。

| **国（社稷）** | 本体。もともと**三種の神器**に入っていた |
| **載り** | その本体に**接続されている**状態 |

| **潮** | **配線**。魔力は海水を伝う |

### 壇ノ浦以降

```
本来：天 → 譲渡 → 神器（国本体）→ 王・依代・社
現在：天 → 譲渡 → 【三種沈没】→ 依代だけが無理やり中継
                                    └ 血に「しるし」だけ残る
```

**載り＝壊れかけの契約。** まだ繋がっているが、本来の器はもうない。

## 母の言葉との重ね読み

> 「**お前は、お前自身の国を、お前の足で探しなさい**」

| 読み | 載りとの関係 |
|------|--------------|
| **A** | 天子・平家の「国」は幻。**載りを名乗るな** |
| **B** | 陸＝組織の世界へ（宇都宮恭順）——**一度は載る** |
| **C** | **しるしを外し**、人として生きる（第3部） |
| **ラスト（物語の芯）** | **自分の中のバッファを外す**＝不思議パワーのない、**みんなと同じ人間の身体**（領地・神の王ではない） |

- 正解は**一つに解かない**（`02_神器たるしるし.md`）  
- 第3部で **A と C が重なる**。B は「一度通ったが、最終的に外した」  
- **竹崎**は言仁のラストに**多く影響しうる**（聞く・動かす・前衛）——`05_竹崎季長_第3部.md`

## 末法と「神話の終わり」（第3部の終焉）★

> **第3部の返天は「末法を終える」ことでもある。** ただし**神仏の徳の回復**ではない。  
> 第2部・第1話の船頭「**末法だ。神も仏も、もう本気じゃねえ**」の**回収**。

### 三層で持つ（表・裏・読後感）

| 層 | 内容 | 使う場所 |
|----|------|----------|
| **表（回線）** | 末法＝神が死んだのではなく、載りの回線が**壊れかけ**で返事が鈍い時代。言仁は修理せず**完全に閉じる**＝末法も**終わる** | 第3部・返天／結 |
| **裏（魂の道）** | **設定奥の層**。神話期は死後に魂が天（社稷の回線）へ**還った**。壇ノ浦・三種沈没で**帰還ルートが詰まり**、古墳・海・怨みに魂が滞る＝末法の正体。返天後は天への回収路が**閉じ**、以後は人の世で**輪廻する**ほかない | **本文NG**（講義しない）。設定整合用 |
| **読後感（人の世）** | 神託は来ない。依代は載らない。だが**畑は実り、子は増え、世は続く**。神に頼らぬ**人間の時代**へ | 第3部・結／エピ |

### 時代の流れ（裏設定・本文に書かない）

```
神話の時代 … 死後、魂は天（社稷の回線）へ還る
壇ノ浦以降 … 三種沈没で帰還ルート故障。魂が地上に滞る（古墳・海・怨み）
末法     … 神仏が本気でないのではなく、行き先が詰まり、祈りも返事も鈍い
第3部の後 … 天への回収路が閉じる。以後は人の世で生まれ、死に、輪廻する
```

- 古墳に潮が溜まる（崇徳の古墳東征）・怨霊が消えにくいのは、**帰還ルート故障**の現れと整合。  
- 「人口がここから増える」は**裏設定の含み**。本文で**人口・輪廻を講義しない**（未来知NG）。

### 本文での出し方（体感のみ）

| 誰 | 描写 |
|----|------|
| **竹崎** | 「末法は、終わったらしい。神も仏も本気を出したんじゃねえ。**返事を、しなくなった**だけだ」 |
| **竹崎** | 「それでも、米は炊けるし、子は泣く。なら、世は終わってねえ」 |
| **描写** | 死んだ者が前より**土に残る**気がする／神託は来ないが**畑は実る**／木花は神と話せないが**馬に乗れる** |

### NG

- 魂・輪廻・人口増加を**地の文で講義**する／未来知（「以後、人は増えた」断定）  
- 末法＝神仏の徳が**回復**したと描く（**閉じた**のであって直したのではない）

## 本文でどう書くか

### NG

- 社稷・プロトコル・載せ替え契約の**講義**  
- 「載り」の**定義説明**をキャラがする  
- ハッキング用語の直書き（→ 竹崎の理解度で曲げる）

### OK（例）

| 誰 | 台詞・描写 |
|----|------------|
| **崇徳** | 「**体が国**」「**載せ替え**を」（短い・余） |
| **言仁** | 「……**侮るな**」（載せ替え拒否） |
| **竹崎** | 「あいつの肩、重い」「**抜く**、ってことか」（粗い言葉） |
| **竹崎** | 「**前みたいに返事が来ねえ**」（返天後） |
| **描写** | 糸・結び目・潮の目・載ってる／外れた |

読者の**崇徳怨霊・幼帝**前理解があれば、用語を一度も説明しなくても、**方向の逆**（崇徳 vs 言仁）で通じる。
