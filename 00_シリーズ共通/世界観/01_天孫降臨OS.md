## 比喩

この世界の「神仏・奇跡・階級・祈祷」は、**天孫降臨OS** と呼ぶ。

| 現実の要素 | 物語での位置づけ |
|------------|------------------|
| 神社 | ルーター（魔力中継） |
| 祝詞 | アクセスキー |
| 依代・ナホビノ | 神と人間のインターフェース |
| クジラ | 海の結節点（サーバー） |
| 神器たるしるし | 壇ノ浦後、血に残る**載りの署名**（→ `02_神器たるしるし.md`） |
| **載り** | 国への**接続状態**（→ `04_載りとは.md` ★） |
| 物部流 | OS以前の裏口。**効果の掛詞**（祝詞の引用ではない） |
| 崇徳・天孫族 | 管理者側のバグ／悪意（死後干渉 → `03_崇徳院と怨載り.md` ★） |

## 物理ルール

- 魔力は**潮の目（海水）**を伝わる。空中には漂わない。
- クジラは潮を固定し、世界のコードを記憶する。
- 鯨油＝燃料ではなく、魔力の記憶媒体。
