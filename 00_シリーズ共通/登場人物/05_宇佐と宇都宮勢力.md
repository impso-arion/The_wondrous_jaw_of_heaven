## 岩戸＝岩門城（地理・原田の夢）★

> 第9〜10話の「岩戸」はすべて**城のこと**。海辺の岩戸神社だけ、ではない。

| 項目 | 内容 |
|------|------|
| **正体** | **岩門城**（呼び名「岩戸」＝この城） |
| **現代** | **福岡県**（筑前・**内陸**寄り） |
| **到達** | **船だけでは着かない**。沿岸（港）まで船→**陸路**で城へ。距離は**けっこうある** |
| **言仁・竹造** | 海で育った者にとって**初めての陸**（馬、道、森、城、畑の匂い） |
| **宇佐** | 大宰府の上。引き渡し・認証は**筑前（岩門城〜宇佐）**の政治圏で動く |

### 原田種直の感情（作者確定）

- 史実どおり、安徳天皇を**岩門城**に迎え、仮御所を造ろうとした男。
- 壇ノ浦のあと、**本当の安徳が、ここに来れる**なら——原田は**泣いて喜ぶ**はず（第10話で露わにしてよい）。
- 平家狩りの義理・宇佐への届け先と**重なる**。冷徹な組織顔の**一瞬だけ**、ゆるんでもよい。**講義・史実解説NG**。
- 第10話・安徳バレの瞬間に、この感情と接続する。

### 第9〜10話との接続

| 話 | 海／陸 | 備考 |
|----|--------|------|
| **9** | **海**（監房） | 「岩戸へ」「岩戸まで」＝**岩門城**への命令名。「航路のうちに答えを出せ」＝**まだ海上** |
| **10** | 海→**陸** | 着岸後、陸路で岩門城。竹造の**初陸**を書く。船で城下に着く描写は**NG** |

## 宇佐神宮（勢力）

### 物語上の位置づけ

- 天孫降臨OSにおける**皇位の外部認証機関**（神託事件のイメージ）。
- 言仁を引き取る**スポンサー**。厳島学園へ送り込む「表の理由」を作る。
- 第2部〜第3部の裏で、**宇都宮の名の下**に人力・領を編成。
- 第6話・定信／教経の説明：**朝廷の上の上**。姓と位を配る（→ `06_第6話`）。

### fiction の役割

| 機能 | 描写 |
|------|------|
| 引き取り | 第1部末。敗北した言仁を「人質」ではなく**御家人**として処理 |
| 姓の授与 | 空いた**宇都宮**を言仁に載せる |
| 学園送り | 四魂の学舎への入学身分 |
| 軍勢 | 第2部は**舞台外**。第3部で宇都宮軍として表出 |

### 登場人物（案・増やしてよい）

| 呼び名 | 役割 | 部 |
|--------|------|-----|
| **宇佐の執政／神領の長** | 言仁に宇都宮を授ける側。冷徹な政治判断 | 第1部末 |
| **神官（名無しで可）** | 「天子の血」を匂い取る。トロイの木馬の送り手 | 第1部末〜第2部（手紙のみ可） |

## 宇都宮朝綱（fiction）

### 奇抜／あこがれ／共通

| 軸 | 内容 |
|----|------|
| **奇抜** | 自分の家を**自ら壊した**武家。生き残りは家老として宇佐に従う |
| **あこがれ** | （あえて置かない）因果の象徴 |
| **共通** | 「一族を失った」痛み——だが言仁に殻を譲る側 |

### 物語上の事実（fiction）

1. 宇都宮朝綱は、内訌・裏切り・過激な決断（詳細は執筆時）で**宇都宮家を空にした**。
2. 領と名だけが残り、誰も継げない。**豊前宇都宮**は看板のみ。
3. 宇佐が言仁にその**空殻の姓**を渡す。朝綱は**家老**として残る（第3部で短く触れる程度）。
4. 第2部本編には**出さない**（噂・「朝綱という名の前当主が…」程度）。

### 史実との距離

- 名前「朝綱」「宇都宮」は借りる。**エピソードは本作オリジナル**。
- 「自分の家を壊した」は史実モチーフの**比喩的転用**でよい。

| 名前 | 誰が使う | いつ |
|------|----------|------|
| **言仁（トキヒト）** | 竹造、村の仲間、銛長時代 | 第1部〜 |
| **宇都宮** | 学園、幕府、公式 | 第1部末〜第3部 |
| **安徳／天子** | 母、崇徳、システム側 | 第8話（母）・崇徳分霊。**原田・読者への露呈は第10話** |

### 言仁の立場

- 宇都宮は**恭順の証**であり、**第3部の軍勢の看板**。
- 「お前はお前の国を持て」と矛盾する——第1部末から第3部までの葛藤。
- 第2部テーマ「利用するな」：言仁自身が**姓という道具**を利用している。

## 第10話：宇都宮（執筆正本）

> 参照：本ファイル ＋ `../../第1部…/構成/01_全10回構成.md` 第10話節

### 必須シーン

| 順 | 内容 | 注意 |
|----|------|------|
| 1 | **海→陸→岩門城**（監房の続き） | 沿岸着岸→**陸路**（初陸）。第9話「航路」の回収。**船で城に着けない** |
| 2 | 原田と言仁の対面。**安徳バレ** | 建礼門院の**面影・声・骨片**など。第8話の母と接続。**講義しない** |
| 3 | 言仁「**降伏する**。人質になれ」 | 全滅回避の**計算**。内面独白なし |
| 4 | 竹造 | 理解できない／でも従う。**竹造**のまま（**竹崎姓は第2部末**） |
| 5 | 宇佐の執政／神官 | 談判。債務・人質・**引き取り** |
| 6 | **離散**（全員宇都宮にならない） | 言仁のみ宇都宮。木花→出雲。竹造→**肥後**。岬の浜の者→**博多移住**（屈辱） |
| 7 | 各国の説明 | 教経・定信が**野望・ネライ**を教える（講義最小限） |
| 8 | 空いた**宇都宮**→**言仁のみ** | **省略不可** |
| 9 | 厳島・四魂 | 入学名：**宇都宮**（言仁のみ） |
| 10 | 竹造・浜・手紙 | 武士を引き連れ浜へ。博多移住を通訳。肥後送り。**言仁・木花への手紙可**。「字が書けるのか」→通訳だから勉強した |
| 11 | 巻末 cliff | 「宇都宮はあいつのだ。俺は竹造のまま肥後」＋手紙／通訳 →第2部 |

### 原田の第10話での役割

- **潰した側**のまま。敵回ししすぎない。
- 宇佐へ**引き渡す**一端。平家の旗を振る側ではない。
- 安徳を知ったうえで、**義理を果たす**（宇佐・平家狩り側への協力）——台詞で説明せず、**結果**で見せる。
- **泣いて喜ぶ**（一瞬でよい）——岩門城に、本物の安徳が来た、と受け取った瞬間。

### 宇佐の第10話での役割

- 敗北した言仁を「処刑」ではなく**御家人化**する処理。
- **宇都宮**という空殻の姓を載せ、学園送りの身分を作る。
- 神官が「天子の血」を匂い取ってもよい（**原田より後**でも可。原田＝気づき、宇佐＝認証、の二段でもよい）。

### NG（第10話）

- 第9話以前に原田が安徳と呼ぶ／断じる  
- 竹造が**竹崎**と名乗る  
- 宇都宮授与の省略  
- 朝綱との対面（第2部以降）  
- 宇佐内部会議・領地政治の本格描写
