### 執筆メモ（現代比喩・作者用）

**リストラされた課長さん**——経営していた会社（出雲の港役・小さな祈祷請負）が潰れ、役目を失い、ジサイとして海に出た。  
娘が地元にいて木花と同じ年齢なので、木花に**親身**になって祈る。「この子を陸に返す」は、自分の娘を返すのと重なる。  
帰りたい思いが強いからこそ、**非効率な命張り**に誠実で、木花の「倍の戒律」も**黙って引き受ける**。

※ 整った執事ではない（髪も梳かず・隔離）。**中身は頼れる年長の係**。

## 史実の芯：持衰（ジサイ）

海を渡る船に同乗する、航海安全を祈る**祈祷師（シャーマン）**。

| 項目 | 内容 |
|------|------|
| 役割 | 嵐・災難を避ける。船内の安全の**全責任** |
| 報酬 | 無事帰港 → 生口・財物などの**豪華なご褒美** |
| 失敗 | 嵐・病 → 「戒律を破った」とされ、**最悪は殺害** |
| 本作 | 史実の過酷さは**残す**。木花の船だけ「家族」に変える |

### 戒律（潔斎）

船に乗っている間——

- 髪を梳かない  
- シラミも取らない  
- 衣服を洗わない  
- 肉を食べない  
- 女性を近づけない  

**ファンタジー解釈**：天孫族が整えた「きれいな世の中」の仕様を**あえて拒う**。文明の利器・社会的秩序を外し、人間本来の**野生**を肥大化させる儀式。  
ジサイの身体が、海と最も繋がりやすい**純粋な状態（依代のポート）**になる。

### 木花流アレンジ

「本来、女は近づけない。でも私が船長（女）だから、**じいじに倍の戒律**を守らせてるの！」

- お気楽回・コメディの燃料  
- 木花の**責任の取り方**（ジサイを守る側）を見せる

## ジサイ術のシステム（たたき台）

### 一句

**「天も地も頼まず、人間の命で海を動かす。」**

- 天孫系：祝詞・神社・龍脈など**外部**のルート  
- ジサイ術：**外部サーバーに頼らない**。個体の精神と肉体を極限まで追い詰め、バーストした生命力で**世界の演算をねじ曲げる**（嵐の止め、船の守り、など）

### 誰が何をするか

| 役 | 担当 |
|----|------|
| **ジサイ（じいじ）** | **安来 定信**。戒律を守り、身を**依代**にする。海と繋がる「受け皿」 |
| **木花** | 歌う・叫ぶ・祈る。**生きたい**という意志をジサイ経由で海へ叩き込む |
| **船** | ジサイと木花の**共同体**。儀式の場 |

### 発動の流れ（執筆用）

1. ジサイは隔離された場所で戒律を守り続けている（普段から）  
2. 危機（嵐・追手・クジラの乱れ）  
3. 木花が**人目を気にせず**祈る（格式より熱量）  
4. ジサイの周囲に**温かい陽炎**のような生気。天降の光とは別物  
5. 効果：嵐の「動き」が一瞬止まる／船だけが守られる／など**一時的**  
6. 代償：ジサイの消耗（本文では「じいじ、顔色が悪い」程度から）

※ 永続的な天候操作はしない。物部流の「最適解」と比べ**非効率**——それが物語の fuel。

## 木花が愛される設定

### ① ジサイ＝使い捨てではなく家族

- 木花がスカウトした**命を預け合う相棒**（年の離れたじいじ）  
- 普段：お転婆・ワイルド。「お前かっこいいから夫になれ！」  
- 海では：「**生きて還るぞ**」という責任  

**愛されシーン（定番）**  
宴のあと、木花が自分の**一番いい取り分**（クジラの肉・ご褒美）を、泥だらけで隔離されたジサイへ人目を盗んで渡す。  
「じいじ、今回も生きててくれてありがと！」

### ② 依代（よりしろ）の拡張

- 木花自身も依代候補だが、**ジサイを前に出す**設計  
- 木花の祈祷が「不思議と効く」＝ジサイのポートを**増幅器**として使う  
- 痛み・消耗は**ジサイに集中**しやすい → 木花の罪悪感・恩返しの動機

### ③ 言仁との思想差

| 言仁 | 木花 |
|------|------|
| 「非効率なバグ技。潮の目を合わせれば足りる」 | 「何が周波数だ！ **生身の命**張ってんだよ！」 |
| 数字・最適解 | 根性・感情 |
| 冷徹 | 泥臭いが**人間味**がある |

**その先（第1部）**  
木花は言仁の孤独（世界の声が流れ込む苦しみ）に**直感**で気づく。  
「終わったら私の船で、じいじのクジラ汁食え。美味いぞ」  
竹崎は宴で見て、「あいつら、良いコンビになるのかも」としみじみ。

## ★ 執筆レギュレーション（ジサイ術）

### 優先順位

1. **木花が誰のために祈るか**（ジサイ／船／仲間）  
2. **ジサイの状態**（戒律・消耗・絆）  
3. 効果（嵐・守り・一時停止）  
4. 言仁との**非効率さ**の対比（説教しない。台詞一つでよい）

### 新場面を足すとき

| やること | 例 |
|----------|-----|
| 祈りの動機を一文 | 「じいじを陸に返したい」 |
| 木花の声・歌・叫び | 格式ある祝詞**ではない** |
| ジサイの身体反応 | 膝が震える／目が潤む |
| 視覚 | 温かい陽炎。冷たい神光と**対比** |
| 代償 | 宴での取り分、または翌日の疲れ |

### やってはいけないこと

- ジサイを**使い捨ての道具**として消費するだけの描写  
- 天孫祝詞と**同じ格式**で木花が唱える  
- ジサイ術で**永久無効化**（言仁級の仕事を奪う）  
- 本文で「サーバー」「OS」等の直書き（竹崎語りで曲げる）  
- 戒律・史実の過酷さを**ごまかしすぎる**（木花の愛があるからこそ重い）
