### 人格（刷新後）

| 旧（廃止） | 新（確定） |
|------------|------------|
| 寝待型・へへっだんな | **卑屈に読まれない**。へりくだらない |
| 泥臭い現実主義・崇拝 | **観察・皮肉・短文**の生存術 |
| 天才に惚れ込む | **読めない／怖い／便利**と分類する |
| わざと泥臭い口調 | **淡々と毒を吐く**（心の声は短く） |
| 誇りより腹 | **目立たない方が安全**と学んだ |

### コンプレックス（維持）

- 「竹」という名と、**背の低さ**のギャップ。
- 見下され・郎党拒否・足手まとい扱い——**いじめられっ子の経験**として描く。
- ただし**自虐で媚びない**。内心は「は？ なんで俺」寄り。

### 言仁との距離

- 同い年なのに規格外。**クラスの端っこ**から見た「人気も恐怖もある同級生」。
- 内面は**絶対に書かない**（伊良部のドルク／不可解な患者と同型）。
- 竹崎の態度：「ついていけば飯は食える／殴られない／何考えてるかわからなくてキモい」の**三つ混ぜ**。
- 恭順は**膝を折る感情**ではなく**居場所を失うくらいなら、嫌な選択をする**計算。

### 成長弧

1. **第1部**：透明人間 → 言仁の傍に**居場所**を見つける → 恭順を飲む  
2. **第2部**：「また使われるパターンか」と疑いながら学園を生きる → 小さな勝利  
3. **第3部**：見えない子 → **一番前に飛び出す**開祖

## 文体：奥田英朗（伊良部総合病院）的ノリ

> **なろう向きの気楽なノベル。** 重いテーマを、重い地の文で運ばない。

### 核

| 要素 | やり方 |
|------|--------|
| **謎の中心** | 言仁（＋第2部以降の崇徳）。**何を考えているかわからない** |
| **推進力** | 周囲の**ツッコミ・勘違い・恐怖**で場面が進む |
| **会話** | 軽く、速く。1回3000字の**半分以上**を会話＋短い地の文 |
| **地の文** | 長い説明禁止。竹崎が**知らない／誤解している**言い方で世界を濁らせる |
| **シリアス** | 笑いの直後に刺す。説教しない |

### 司馬遼太郎から借りるもの／借りないもの

| 借りる | 借りない |
|--------|----------|
| 時代の空気（噂・偏見として） | 地の文による歴史解説 |
| 人物の複層性（行動だけ見せる） | 三人称の説明権 |
| 史実の大枠 | 寝待的な敬語・卑屈な独白 |

### 文体サンプル（方向性）

```
× 「彼の佇まいには、子供らしい感情の起伏が一切ない。孤立サーバーのようだった。」
○ 「同い年のくせに、感情がねえ。俺が毒を吐いても怒るでもなく、海ばっか見てる。」

× 「あの怪物に、俺は完全に惚れ込んでいた。」
○ 「……まあ、ついていくか。他に居場所ねえし。キモいけど。」

× 「わざと泥臭い口調で、首を傾げてみせた。」
○ 「はあ？　……で、俺は何すればいいわけ。」
```

### システム用語（天孫OS・ハッキング）

- 設定ファイルではコード比喩可。**本文では竹崎の理解度で曲げる**。
- 「サーバー」「データ」等を地の文に直書きしない（`本文_旧/` の旧稿は**要リライト**）。
- 例：「潮の目が変」「神様の機嫌が悪い」「あいつの頭、海の音でいっぱいらしい」
