## シリーズの骨格（一行）

10歳の銛長・言仁が、敗北と恭順の末に**空いた「宇都宮」の姓**を宇佐から受け取り、学園（第2部）を内側から解析し、20歳で超常を消して**竹崎（人間）に世界を明け渡す**。

## 第1部：言仁と死者のクジラ（10歳）

### 巻テーマ
楽園は組織に勝てない。**敗北・支援・恭順**。名字まで預ける。

### 起承転結

| 段 | 事件 | 回 | 備考 |
|----|------|-----|------|
| **起** | 捕鯨船。竹造と言仁の出会い。物部流の一撃 | 1〜2 | プロローグ執筆済 |
| **承** | 岬の村・沖ノ島・木花合流・凪の日 | 3〜5 | 執筆済パートあり |
| **転** | ジサイ・**原田／宗像**・死者のクジラ・母 | 6〜8 | |
| **結** | 一番モリ（**原田組織**）→離散→**宇都宮授与** | 9〜10 | |

**回単位詳細** → `../../第1部_言仁と死者のクジラ/構成/01_全10回構成.md`  
**敵組織** → `../../第1部_言仁と死者のクジラ/素材/人/01_原田種直と宗像勢力.md`

### 巻末：離散と「宇都宮」

1. **一番モリ紛争**  
   死者のクジラ破壊＝システム損害。正規軍・大国水軍が介入。言仁は「降伏・人質化」を選択（全滅回避）。

2. **スポンサー引き取り**  
   | 人物 | 行き先 | 役割 |
   |------|--------|------|
   | 言仁 | **宇佐（豊前）** | 皇位認証の外部機関の庇護下 |
   | 木花 | 出雲 | 依代として囲う |
   | 竹造 | 肥後 | 魔術から最遠。人間の前線 |

3. **空いた宇都宮**（巻末クライマックス）  
   - **宇都宮朝綱**は、一族の内訌（fiction）で**自分の家を壊した**。家名だけが空いた。
   - 宇佐の判断：「人質の少年より、**御家人・宇都宮**の方が厳島の学園にも通せる。」
   - 言仁は**表向きの身分**として**宇都宮**を名乗ることになる。  
     本名・銛長としての「言仁」は、竹造など内側の者だけが使う。
   - 朝綱本人：家を壊したあと、**宇佐の下で家老／残党**として残る（第1部末に影として一度触れる程度。詳細は04参照）。

4. **言仁の内面（巻末状態）**  
   - 天子の血を知る。母の「自分の国を持て」と矛盾し始める。
   - **宇都宮**＝恭順の証。自分の国ではない、組織が渡した殻。

### 第2部へのバトン
- 厳島・四魂の学舎へ。**入学名：宇都宮（言仁）**。
- 竹造は肥後、木花は出雲経由で後から学園合流。

## 第2部：言仁と神器の鳥居（15歳）

### 巻テーマ
他者を利用するな。**怒りと小さな勝利**が学園OSを狂わす。

### 舞台の原則（厳守）

> **第2部は厳島・四魂の学舎が中心。外部政治は本編に入れない。**

| 書いてよい | 書かない |
|------------|----------|
| 学園内の対人関係・魔術・崇徳 | 豊前での領地会議 |
| 「宇都宮殿」という肩書・偏見・噂 | 朝綱との直接交渉 |
| 届いた手紙の一行（宇佐から） | 宇都宮軍の動員シーン |
| 北条・木花・竹崎の学園内ドラマ | 幕府と宇佐の全面外交 |

※ 宇都宮の軍勢が育つのは**第2部の舞台外（裏）**。第3部で初めて表に出る。

### 言仁と「宇都宮」（第2部での扱い）

- 同学年からは**九州の殿様・宇都宮**として見られる。
- 竹崎の独白：「あいつ、銛長の頃と顔つき同じなのに、いつから**殿**なんだ。……キモいけど、ついていくか」
- 言仁は姓を**道具**として使う。倫理のグレー（第2部テーマ）の一端。

### 第3部へのバトン
- 崇徳の牙城を内側から喰らう包囲網完成。
- **宇佐経由で、第2部の裏で集めた宇都宮の人力**が、第3部の物理兵力になる。

## 第3部：言仁と黄泉の山頂（20歳）

### 巻テーマ
言仁が超常を消し、**竹崎（人間）に世界を明け渡す**。

### 兵力の三層（fiction）

| 層 | 勢力 | 役割 |
|----|------|------|
| **中央** | 鎌倉幕府（北条） | 第2部の裏同盟。正規のパッチ |
| **地方** | **宇都宮**（宇佐の網） | 言仁が姓で束ねた**人間の軍勢** |
| **人間** | 竹崎（肥後） | 決戦の前衛 → 蒙古前線の受け皿 |

### 起承転結

| 段 | 事件 |
|----|------|
| **起** | 20歳。竹崎の家。宇都宮**言仁**としての再会 |
| **承** | 霧島へ。宇都宮軍・幕府軍が動く（ここで初めて軍勢を描く） |
| **転** | 天孫返天の儀。**しるし摘出**。崇徳パージ。依代消滅 |
| **結** | 言仁は**人として**去る（死なない）。超常消失 |
| **エピ** | 蒙古。神風なし。竹崎（一族）の突撃 |

### 言仁の結末（確定）

- **死なない**。自ら**神器たるしるし**を抜き、**人として生きる**。  
- 平家物語的「安徳の死」で締めない——**自分で皇室の載りを外す**別結末。  
- 宇都宮看板は剥落。**歴史から消える**（隠棲）。  
- 詳細 → `../../第3部_言仁と黄泉の山頂/設定/01_言仁の結末と神器たるしるし.md`  
- 世界観 → `../世界観/02_神器たるしるし.md`

### 宇都宮の結末（fiction）

- 決戦後、**宇都宮**という看板は歴史から剥落。
- 朝綱：第3部で短く——家を壊した男が、空けた殻を他人に預けた報い、程度でよい。
- **竹崎**が肥後に「自分の国」を持つ。第1部母の言葉の回収。
