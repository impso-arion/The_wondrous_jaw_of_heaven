## シリーズ全体を貫く一本の線

**「超常の力でできたシステムは、いずれ人間の手から外れなければならない」**

- 表の物語：言仁（安徳）と竹崎（ただの人間）が、神仏・魔術・階級という**天孫降臨OS**を解体していく。
- 裏の物語：三部を通じて、**言仁が力を集め、竹崎に日常を返す**。
- **語り手**：主に**竹崎季長（竹造）**一人称。言仁の内面は書かない（伊良部の不可解な中心人物と同型）。
- **文体**：**奥田英朗（伊良部総合病院）的**——軽妙・会話多め・謎は謎のまま進む。詳細 → `../登場人物/05_竹崎季長と語り口.md`
- 竹崎の内面：**令和いじめられっ子 sensibility**（卑屈・寝待型は廃止）。恭順は**計算された生存**。

## 第1部：言仁と死者のクジラ

### 巻テーマ（確定案）

**自由と孤独と無法者たちの楽園は、より大きな組織に勝てない。  
敗北と支援と恭順——生き延びるために、恭順と支援を受け入れる現実。**

| キーワード | 意味 |
|------------|------|
| **自由** | 岬の村。長も地頭もいない。流れ者の楽園 |
| **孤独** | 言仁の隔絶。竹造の背の低さ。誰も本当の言仁を知らない |
| **無法者の楽園** | クジラと米。飢えから解放された「天国」 |
| **より大きな組織** | 宗像・鎌倉・正規水軍・天孫OSの法 |
| **敗北** | 一番モリ紛争。死者のクジラの代償 |
| **支援** | 宇佐・出雲・肥後が引き取る。スポンサー＝生き残りの条件 |
| **恭順** | 言仁の「降伏せよ」。人質・養子として組織の懐へ |
| **宇都宮** | 空いた家名を**姓として預ける**。自分の国ではない殻 |

### この巻で読者に感じさせたいこと

- 楽園は**永続しない**。勝てば勝つほど、大きなシステムの目に止まる。
- 恭順は屈辱だが、**全滅よりマシ**——言仁はそう選ぶ。竹崎は後から「また使われる側か」と理解する。
- 「お前はお前の国を持て」は、**組織の王座ではなく、自分の足で立て**という母の言葉。第1部の感情的な着地。

### プロットとの対応

| 場面 | テーマの機能 |
|------|--------------|
| 岬の内の村 | 楽園の提示。自由の上限を見せる |
| 死者のクジラ・母 | 個人的な愛。組織を超えた動機（巻内完結） |
| 一番モリ紛争 | 組織との衝突。敗北の宣告 |
| 離散・厳島行き | 恭順・支援・人質化。現実の厳しさ |
| 宇佐・宇都宮授与 | **名字まで預ける**恭順の極致。厳島入学の身分 |

### 巻末の状態

- 言仁：天子の血を知る。**宇都宮**を名乗る。組織への内部潜入。
- 竹造：肥後へ。恭順を飲み込んだが、**生き延びた**。
- 木花：出雲へ。依代として囲われる。

## 第2部：言仁と神器の鳥居

### 巻テーマ（確定案）

**他者を利用してだますことを行ってはならない。  
「生身の怒り」と「泥臭い小さな勝利」が、完璧だったはずのシステムをガタガタに狂わせていく。**

| キーワード | 意味 |
|------------|------|
| **利用と欺瞞（禁則）** | 言仁の冷徹な最適解（木花を「泳がせる」等）が倫理的な試金石になる |
| **生身の怒り** | 北条泰時、木花、竹造の感情。コードでは測れない熱 |
| **小さな勝利** | 魔術なしの刀一本、小指の痺れ、学園での武功、「竹崎」の名 |
| **完璧なシステム** | 四魂の学舎・崇徳校長・鳥居（人間と神域の境界） |
| **ガタガタ** | 依代誘拐、禁断同盟、内部からのバグ拡大 |
| **宇都宮（学籍）** | 肩書と偏見のみ。**外部政治は書かない** |

### この巻で読者に感じさせたいこと

- 言仁の「正しい計算」は、**人を道具にする**ところまで行く。それが対価を要求する。
- システムは整然と見えても、**一人の怒りと一点の勝利**で亀裂が入る。
- 神器の鳥居＝**くぐればくぐるほど、利用する側／される側の境界が曖昧になる**。

### プロットとの対応（案）

| 場面 | テーマの機能 |
|------|--------------|
| 厳島入学・再会 | 組織の内部。鳥居の内側 |
| 崇徳の生贄システム | 「完璧なOS」の裏のバグ |
| 木花誘拐 | 言仁の冷徹プラン vs 他者の怒り |
| 竹崎の武功・「竹崎」授名 | **小さな勝利**の結晶 |
| 鎌倉との裏同盟 | 組織を組織で壊す。倫理の灰色 |

### 巻末の状態

- 言仁：崇徳の牙城を内側から喰らう決意。まだ**力は集め途中**。
- 竹造：人間としての勝利を一つ手にした。言仁への信頼と疑念が共存。
- 木花：さらわれたまま、またはシステムの核心へ。

## 第3部：言仁と黄泉の山頂

### 巻テーマ（確定案）

**組織と力を持った言仁が、システムごと超常の力を消し去り、  
竹造（ただの人間）に世界を明け渡す。**

| キーワード | 意味 |
|------------|------|
| **組織と力** | 宇佐・**宇都宮の軍勢**・幕府同盟・物部流・全データ |
| **超常の力の消去** | 天孫返天。神仏・奇跡・依代・魔力回路の完全停止 |
| **明け渡す** | 言仁は**しるしを自ら抜き**人として生きる。歴史から消える。勝者は**竹崎** |
| **神器たるしるし** | 壇ノ浦後、血に残る**載りの署名**。第3部で**自ら外す** |
| **黄泉の山頂** | 天孫返天＋摘出の地点。神域の最奥 |

### この巻で読者に感じさせたいこと

- 神を殺したのは**英雄の凱旋ではない**。言仁は**死なず**、載りだけを外す。
- 蒙古襲来に神風が吹かない＝**人間だけの戦い**が始まる。竹造（一族）が受け取る世界。
- 三部のテーマ回収：第1部の恭順で生き延びた者が、第3部で**自由な世界**を受け取る。

### プロットとの対応（案）

| 場面 | テーマの機能 |
|------|--------------|
| 霧島・天孫返天 | 超常の力の総消去 |
| **しるし摘出** | 皇室の載りを**自ら外す**。平家物語的悲死を回避 |
| 崇徳の完全敗北 | システム悪の終焉（力ずくのパージ） |
| 言仁の隠棲 | **人として**去る。託す |
| 蒙古・竹崎の突撃 | **ただの人間**の世界の開幕 |
| 宇都宮軍の動員 | 第3部で初表出。姓の看板が**人間の兵力**になる |

### 巻末の状態

- 言仁：**しるし摘出**。**人として生きる**。歴史から消える（再登場しない／転生確定はしない）。
- 竹造／竹崎季長：開祖。神に頼らない戦いの起点。
- 世界：奇跡のない現実の海と陸。

```
第1部  楽園 → 敗北 → 恭順・支援        「組織に勝てない現実」
         ↓
第2部  怒り → 小さな勝利 → システム亀裂  「人間の熱がOSを狂わせる」
         ↓
第3部  言仁が力を集めて神を消す → 託す  「超常を消し、人間に返す」
```
