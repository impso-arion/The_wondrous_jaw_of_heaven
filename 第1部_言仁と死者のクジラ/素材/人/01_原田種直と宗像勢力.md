## 史実メモ（執筆の芯）

### 原田種直（はらだ たねなお）

| 項目 | 内容 |
|------|------|
| 生没 | 1140–1213頃（治承・寿永期の現役） |
| 地位 | **大宰権少弐**（1181頃）。筑前・**岩門城（岩戸）**を本拠 |
| 平家 | 平清盛・重盛と主従。**重盛の養女を妻** |
| 安徳 | 都落ち後、**岩門城**に仮御所を造り、安徳天皇を迎え入れようとした（本作＝**福岡・内陸**の城） |
| 壇ノ浦後 | 捕縛→赦免（1190頃）→**怡土（糸島）**に復帰 |

※ 本作では言仁が10歳・周防長門にいるのは**fiction**。種直が安徳と**断定**するのは**第10話**（建礼門院の面影など）。第7〜9話は「密猰／物部の裏口」程度。

## 登場人物（fiction）

### 原田種直（本人または「声だけ」）

| 軸 | 内容 |
|----|------|
| **奇抜** | **岩門城**に仮御所を造った男。妻が平家の血。**安徳が来れるなら泣いて喜ぶ** |
| **あこがれ** | （置かない）組織の顔 |
| **共通** | 「秩序を守りたい」——悪意より**体制側** |

**物語上の動機**
- 壇ノ浦後、**平家狩り側**の義理。安徳の血を**宇佐**へ届ける（台詞で説明しない）。
- 密猰・物部流・死者のクジラ破壊＝**大宰府の損害**。
- 言仁を**生かして**岩門城まで運ぶ（第9話・海上の監房→第10話・**陸路**護送）。

**第1部での出方**
- **7話**：差し向けの侍の背後に「大宰権少弐・原田」の名。大船団・白旗。
- **9話**：本人初対面。賊害・**岩戸（城）**・宇佐の命令。**縄を解き監房**（まだ海）。
- **10話**：着岸→**陸路**→岩門城。安徳露呈→降伏→**宇佐へ**（詳細 → `05_宇佐と宇都宮勢力.md`）。

## 駒としての配分（第10話・正本）

> 教経の「**駒**」。言仁だけでなく**竹造・木花**も政治的に配分される。  
> キャラ別詳細 → `00_竹崎季長.md`・`02_木花.md`・`01_言仁.md`

| 誰 | 組織の見方 | 行き先 |
|----|------------|--------|
| **言仁** | 安徳の血・宇都宮の殻 | **宇佐**（厳島・四魂） |
| **木花** | 依代・船団の**駒** | **出雲**（配分——完全な自由ではない） |
| **竹造** | 通訳・郎党・人質網の**駒** | **肥後**（言仁側に**付けない**） |
| **頼舟** | 言仁の実務・**付き従** | **言仁に付く** |
| **与市** | 乗組 | **別の船** |
| **岬の浜** | 密猰共同体 | **博多**移住（竹造が通訳） |

- 原田「**郎党も連れろ**」の**逆**——竹造は**別枠**で肥後。
- 第9話「甘い」裁定＝生きた人質を宇佐／岩門城へ（第10話で露わ）。
