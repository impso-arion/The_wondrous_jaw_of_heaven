| 項目 | 内容 |
|------|------|
| **呼び方** | 本人→**教経**。竹崎→**おっちゃん**／育ての親 |
| **年齢** | 年長（50前後） |
| **出身** | **奈良**（佐保の里寄り） |
| **経歴** | **平家方**の下級武士出身。若き日、**平清盛**に謁見したことがある（誇りではなく、生き延びの話として語る） |
| **現在** | 言仁の「陸の器」。主張は**育てた**ではなく「用意した」 |
| **口調** | **落ち着いた老人**。諺・比喩多め。「～じゃ」「～のう」。説教は長くしない |
| **例** | 「主は肩書きじゃない。あの御方が『こうしろ』と言った瞬間、村は動くのう」 |

## 言仁の船 — 疲れた三役（親方・網師・炊き）

> **木花合流前**は、全員が**戦争と海に削られた**温度。笑いは薄い。怒鳴りは**癖**で、元気ではない。  
> **木花合流後**（第4話末〜第5話）：同じ台詞でも**明るさが増す**。宴の火が大きくなる。——**木花の熱**が、船を照らす（説明せず、会話と仕草で見せる）。

### 岩松 親彦（いわまつ ちかひこ）— 親方

| 項目 | 内容 |
|------|------|
| **呼び方** | 乗組→**親方**。本名は**あとから**（第3話会合などで名乗る） |
| **年齢** | 50前後 |
| **役割** | **船の持ち主**。帆・桅・出港の最終責任。頼舟と役割分担 |
| **疲れ** | 戦で港を失い、船だけが残った感じ。**肩が常に下がっている**。褒めない。呻く |
| **木花後** | 呻きは減る。木花の冗談に、**小さく吹き出す**程度。出港の声は、少しだけ上がる |
| **口調** | **重い・短い**。「～だ」「～だろうな」。諦めと実務の混合 |
| **例（前）** | 「……またか。潮の目が変だ」／「明後日、出港だ。終わらなくていい」 |
| **例（後）** | 「……二艘で挟めたな。米、白くできるかもしれん」 |

※ **岬の内の「大工の親方」**は別人（村の大工長）。混同しない。

### 網代 弁助（あみしろ べんすけ）— 網師の爺

| 項目 | 内容 |
|------|------|
| **呼び方** | **爺**／**網師**。竹崎→背の低い、と呼ぶ |
| **年齢** | 60前後 |
| **役割** | 網・縄・端の仕事の**目**。引き綱の段取り |
| **疲れ** | 舌が荒いが、**怒りより疲れ**が先。戦争で網師仲間が減った。若い奴への当たりは**習慣** |
| **木花後** | 舌はまだ荒い。**「動け」のあと、肉を一枚よこす**など、照れ隠しの優しさ。木花に「爺、網、強く」と言われると黙って従う |
| **口調** | **ぶっきらぼう・短文**。「～しろ」「～じゃ」。敬語なし |
| **例（前）** | 「背の低い、端、持て」／「バカな話の船に、バカみたいに乗ったな」 |
| **例（後）** | 「……食え。動く奴に、網は回る」／「木花殿の船、網がいい。……別に、感謝してねえ」 |

### 釜田 お先（かまだ おさき）— 炊きの女

| 項目 | 内容 |
|------|------|
| **呼び方** | **炊きの女**／**お先さん**（与市・木花が使う） |
| **年齢** | 40前後 |
| **役割** | 米・汁・脂の**釜**。腹を膨らませることが戦い |
| **疲れ** | 寡黙。**神の名前**を連呼するのは、怖さと習慣。笑顔はほとんどない。指は荒れている |
| **木花後** | 木花が「ご褒美に肉」と言うのを、**真似して**小石を投げる（第5話）。宴で**汁をすする音が大きい**＝嬉しい。打毬の負け組の炊事で、**口数が増える** |
| **口調** | **寡黙→木花後は少し増える**。基本は短い。「～だろう」「～ねえ」 |
| **例（前）** | 「眼、だけ」／（台詞少ない。釜の音で存在感） |
| **例（後）** | 「負けた方が、炊くのねえ。——多めに、味付けするわ」／「木花殿、米、もう一合よい」 |

## 木花合流 — 船の温度（執筆メモ）

| 時期 | 言仁の船の空気 |
|------|----------------|
| **第1〜3話** | 疲れた三役＋冷たい頼舟＋沈む教経。与市だけが**やや明るい**（若いから） |
| **第4話・合流** | 木花・潮風・春風の**声と歌**が甲板に乗る。宴で**お先の釜が賑わう** |
| **第5話** | 打毬・特訓。**お先が石を投げる**。弁助が肉をよこす。親彦が笑う |

**やらないこと**：地の文で「木花のおかげで皆ハッピー」と説明しない。  
**やること**：台詞の数・音量・仕事の速さ・宴の描写で、**前後を見せる**。

## 木花の船 — 潮風・春風

> **ジサイ（安来 定信）の娘ではない。** 定信の娘 **梅** は出雲・安来にいる（`../../../00_シリーズ共通/登場人物/04_木花とジサイ.md`）。

### 潮風（しいかぜ）

| 項目 | 内容 |
|------|------|
| **年齢** | **25** |
| **役割** | 木花の船の**操船・帆**。パワフル |
| **経歴** | **小さな帆船を一人で操れる**。船とは何かを**木花にゼロから教えた** → 木花は船に詳しい |
| **木花へ** | **妹**のように可愛がる＋**呪力持ちの船長**として畏怖 |
| **口調** | **太く豪快**。「～だろ」「～しな」。実務的で速い |
| **例** | 「帆は風を読むんじゃねえ、風に乗るんだ」／「木花、投げろ。曲げるのは俺が教えただろ」 |

### 春風（はるかぜ）

| 項目 | 内容 |
|------|------|
| **年齢** | 潮風よりやや若い（20前後） |
| **役割** | 網・補給・**見た目の整え** |
| **信念** | **女らしさ**に誇り。**美しい＝男を黙らせる尊厳**（教えられた） |
| **漁船** | 甘えはない。男の多い世界で**惑わせる可能性**はある |
| **定信の小言** | 木花に**悪影響がないよう**、色っぽい振る舞いは**我慢しがち** |
| **木花へ** | 潮風と同様、妹＋船長への畏怖 |
| **口調** | **柔らか・品がある**。「～ですわ」「～かしら」。ただし芯は硬い |
| **例** | 「網端は汚くても、手元は汚さないの」／「……ジサイ様、見てらっしゃるわよ」 |

### 五島 銛介（ごとう せんすけ）— 網組の頭

| 項目 | 内容 |
|------|------|
| **呼び方** | 乗組→**銛介**／**網頭**。敵側から→**五島の** |
| **年齢** | 30前後 |
| **立場** | 西海を荒らす**大規模捕鯨集団（網組）**のリーダー。無法者 |
| **戦法** | **ジサイを何人も**船に積み、祈祷を**オーバークロック**（命を削り切って海を歪める）。クジラを座礁させる／潮を乱す |
| **木花との関係** | **許せない敵**。定信一人・命を削る戒律を守る木花にとって、ジサイを**消耗品**にするのは冒渎 |
| **言仁との関係** | 物部流の「純粋なコード」に目をつけ、奪って自系統を強化したい**歪んだ生存哲学** |
| **口調** | **高圧・短い**。成功体験のある男。「～だろう」「～しろ」。敬語なし。人を見下す |
| **例** | 「神託？　警告？　出力を上げれば、握りつぶせる」／「ジサイが足りねえなら、積め。海は、壊れてから直せばいい」 |
| **竹崎の目** | 「アイツの船の近くにいると、頭が痛い。クジラを狩ってるんじゃねえ、**海をなぶり殺し**にしてる」 |

**執筆メモ**：システム語直書きNG。竹崎は「頭が痛い」「人数のジサイ」「唱えすぎ」で理解する。

### 宗像 漣（むなかた さざなみ）— 正規の水軍

| 項目 | 内容 |
|------|------|
| **呼び方** | 本人→**漣**。部下→**漣殿**／**宗像の若いの** |
| **年齢** | 20前後 |
| **立場** | **宗像水軍**の若き統率者候補。**原田種直**差し向けの顔（大宰府・平家圏の海の正規） |
| **戦法** | **正規の神の恩寵**——整った祝詞・お祓い・宗像大社系の「海上航行コマンド」。物部流の裏口を**違法（密猰）**と見なす |
| **言仁との関係** | **ライバル**。好戦的だが、銛介ほど狂気ではない。**美しい正規**のプライド。言仁の最適解に揺れる余地（第1部では浅く） |
| **口調** | **格式・上品**。短めだが、**です・である**調。「～いたします」「～でございましょう」 |
| **例** | 「貴殿らの狩りは、神託に背く密猰。退け」／「宗像の海は、祝詞で整える。蛮行は、不要です」 |
| **竹崎の目** | 「綺麗な服着て、上品にクジラを呼びやがる。——俺たちを、**バグ**扱いかよ」 |

**執筆メモ**：第3話の神託（取るな）と**同じ側**の顔。だが言仁グループを**潰す**立場でもある。漣＝「正規の怒り」、銛介＝「無法の狂気」の対比。
