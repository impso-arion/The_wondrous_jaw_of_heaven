### 第1話：潮合の調律

**本文** → `../本文/01_第1話_潮合の調律.txt`

#### 捕鯨の手順（第1部の型・本話で初出）

| 段 | 誰が | 何をする |
|----|------|----------|
| **①発見** | 銛長・船頭 | 潮合の音・潮目の変化で巨体の接近を察知。パニックの中でも銛長の一言が甲板を動かす |
| **②一番モリ** | 銛長 | 呼吸のための**浮上位置**を読み、潮合に合わせて**尖刺**（物部流・唱えあり）。急所への第一刺 |
| **③打ち銛** | 銛役 | 縄付き銛を投げ、皮膚に喰い込ませて引き綱を張る。巨体を船側に固定する |
| **④とどめ** | 銛長 | 二度目の尖刺（唱えなし）。**命の急所**——咆哮が途切れる |
| **⑤けん引** | 網引き・全員 | 船で引き、**岬の内の浜**まで運ぶ。船上解体はしない（全員海に落ちる） |
| **⑥解体** | 村共同体 | 砂浜で脂・肉を分け、米と交換する——**第2話へ** |

> 竹崎は順番を理解していないまま「動け」「引け」だけ言われる。**手順の全体像は読者にも段階的に見える**ようにする。

#### 言仁のすごさ（本話の核）

| 項目 | 内容 |
|------|------|
| **呼吸位置** | クジラが息を吹き上げる場所を、**水中がすべて見えるかのように**読む。右舷・十五間・「息する」——見えないのに正確 |
| **潮合** | 吐息のリズムを波の間に合わせる。**調律**＝本話タイトルの意味。尖刺はその合図の上で走る |
| **船の反応** | 「銛長の眼だけ、信じてる」（炊きの女）。潮干狩りの日＝銛長がいない日。米が白いのは銛長がいるから——**バカな話を真面目に信じる船** |
| **安徳の秘密** | 大人の**一部**だけ、彼が安徳であることを知っている。それ以外は「神話的な存在」という**畏怖**から盛り立てる——正体は語らない |
| **竹崎の距離** | 怖い／便利／考えは読めない。**態度（肩・悔し）の芽**——本格は2話〜（`00_竹崎季長.md`） |

| 段 | 内容（本文＋厚み） |
|----|---------------------|
| **起** | 竹造、米のため乗船。「バカな話」の海（山みたいな魚）。網引き役に割り当てられる。銛長（言仁）——同い年十歳、体だけ大人二回り、海ばかり見る異物。「銛長がいねえ日は潮干狩り」 |
| **承** | 船の役割分担（船頭・炊き・親方・網代）。潮の目が変だ——銛長が静かなら近い。言仁「魚じゃない。クジラだ」。みんなは**眼だけ**信じる。竹造はまだ態度までは見ていない |
| **転** | クジラ登場・パニック。言仁「右舷、十五間。息する」→潮合→**一番モリ**（尖刺・唱え）。打ち銛→**とどめ**（二度目・命の急所）。与市・頼舟・弁助が各役で動く |
| **结** | 「網を引け、竹崎」。**けん引**——岬の内の浜まで。解体は砂浜（第2話）。降りる港はもうない気配 |
| **崖** | ついていけば飯は食える／何考えてるかわからない——**第1部通りの距離の芽** |

| 項目 | メモ |
|------|------|
| **登場** | 竹造、言仁、頼舟、与市、弁助、炊きの女、親方 |
| **世界観** | 潮合・尖刺・海の「バカな話」。捕鯨＝**六段の共同体技**（発見→モリ→打銛→とどめ→けん引→解体） |
| **言仁** | 呼吸位置の正確さ＝**水中透視に近い**。畏怖と「眼」の信仰。安徳は一部のみ |

### 第2話：岬の内の天国

**本文** → `../本文/02_第2話_岬の内の天国.txt`

#### 解体の共同体（第1話⑥の続き）

| 段 | 誰が | 何をする |
|----|------|----------|
| **①着岸** | 全員 | 第1話のけん引の続き。浜に巨体を引き上げる |
| **②役割分担** | 村全体 | 皮・脂・骨・油。子供が重労働、大人が切り方・保存を判断 |
| **③言仁の一声** | 銛長 | 海を見たまま短い指示——大人が一瞬で動く／すぐ揉める |
| **④通訳** | 竹造 | 揉めを並べ直して届ける。**本話の技の初出** |
| **⑤食事** | 共同体 | クジラ汁と白米。**天国**＝米が本当、の証明 |

> 長も地頭もない**かりそめの村**。格式より「足」——言仁の一声で回る。第1話の海の技に対し、本話は**陸の技**（解体・分配）。

#### 本話の核：通訳の芽

| 項目 | 内容 |
|------|------|
| **言仁の指示** | 結節語（骨／油／皮／船）。陸と海の音が混ざり、誰向けか不明 |
| **竹造の仕事** | 並べ直して届ける。嘘はつかない。一番わかりやすかったのは**自分の口調**——少し怖い |
| **教経** | 「**曲がった意味をお前が通せ**」。主は言仁のまま。勘違いするな |
| **格式 vs 足** | 肩書きの主はいない。動くのは言仁の足と、届ける竹造の口 |
| **大人 vs 竹造** | 教経は「**御方**」、与市は「銛長」。竹造だけ「**言仁**」と名で呼ぶ |

#### 言仁の気づき（郎党の前）

| 項目 | 内容 |
|------|------|
| **大人** | 御方・眼・畏怖。言葉を**拝む**／争う |
| **竹造** | 銛長／言仁。言い直せるか**聞く**。神託ではなく**話者**として向き合う |
| **気づき** | 同い年が、海に逃げず**こちらを見た**。名で呼ばれた——**内面は書かず、態度だけ** |
| **「よく聞こえる」** | 通訳の有用性＋**自分に言葉が返る**（二層。地の文で説明しない） |
| **「なら、よかった」** | 格式不要。**竹造が来てくれた／残る**——契約論の奥 |

#### 郎党志願（格式のすれ違い）

| 段 | 誰 | 内容 |
|----|-----|------|
| **志願** | 竹造 | 「郎党にしてくれ」——格式・縁・離れない（与市には説明が要る） |
| **拒否** | 言仁 | 「**郎党は断る**」——**格式**としての名付けを拒む（人間を拒むわけではない） |
| **言い直し** | 竹造 | 郎党を別語彙に翻訳（現行：「契約しろよ」——**竹造の台詞**。言仁ではない） |
| **すれ違い** | 言仁 | 字面で受け取る。「ずっと助けるのか」→竹造「契約しなくても助ける」 |
| **結び** | 言仁 | 「**なら、よかった**」——①の気づきの上で、竹造が残る |
| **笑い** | 周囲 | 十歳に郎党／背の低いの大物気取り。教経だけ笑わない |
| **着地** | 竹造 | 「**勝手についてる**」。郎党なしでも居場所は消えない |

| 段 | 内容（本文＋厚み） |
|----|---------------------|
| **起** | 岬の内着岸。解体⑥の本番。長・地頭なしのかりそめの村。子供も働く。教経「主はあの御方だけ」——格式より足 |
| **承** | 包丁役・脂剥ぎ。言仁の手が支える。共同体の熱。白米とクジラ汁——**天国** |
| **転** | たき火前の指示が揉める→竹造が通訳→**言仁の気づき**（海に戻らない／名で呼ぶ）→教経が役割を定義→郎党志願・拒否・「よく聞こえる」→すれ違い→「なら、よかった」→笑い |
| **结** | 「勝手についてる」。五歩後ろに戻る。居場所の芽（口にしない） |
| **崖** | 「明日、会合がある。来い」——郎党じゃないのに呼ばれる。潮の向こうの島の影（**沖ノ島**） |

| 項目 | メモ |
|------|------|
| **登場** | 竹造、言仁、教経、頼舟、与市、弁助、お先、親方、刀鍛冶、炊きの女 |
| **世界観** | 楽園＝クジラと米。解体共同体。格式のない村が銛長の足で回る |
| **言仁** | 陸の結節語・口下手。郎党格式は拒否。「おまえがいると、よく聞こえる」 |

### 第3話：沖ノ島の警告

**本文** → `../本文/03_第3話_沖ノ島の警告.txt`

> 第2話の**陸の通訳**が、本話で**商いの通訳**へ拡がる。楽園（天国）の外側に、**正規（宗像・朝廷の法）**が初めて本気で立つ。

#### 彦島商い（出航〜売買）

| 段 | 誰が | 何をする |
|----|------|----------|
| **①会合** | 村全体 | 明後日の明・出港。行き＝彦島、帰路＝玄界灘で漁。**竹造がまとめる**（第2話の続き） |
| **②積荷** | 全員 | 骨・油・銛。浜の解体は未完でも「売れる分だけ」——言仁の結節 |
| **③売買** | 言仁＋竹造 | 言仁「**油は、岸に下ろすな**」「**骨を、岸に置け**」——商人激怒 |
| **④通訳** | 竹造 | 結節を商い語に直す（骨に油の値をのせる等）。蔵を比較し条件交渉。**通訳の実戦** |
| **⑤成果** | 共同体 | 白米・鉄・網の麻が戻る。言仁は金の数を見ない |

> 漁だけでは網も直らない——**楽園は商いにも依存**する。言仁の結節は陸の金の文法とズレる。届けるのは竹造。

#### 玄界灘への航路（沖ノ島の作法）

| 項目 | 内容 |
|------|------|
| **計画** | 彦島で荷下ろし→玄界灘へ。**本番のクジラ**は帰路の潮目 |
| **沖ノ島** | 女神の島。左舷に見えたら**供物を流す**——通過のみ、**寄港しない** |
| **宗像のルール** | 正しい祈祷を唱える船だけが通れる。怠れば**水軍の口実**（教経） |
| **言仁** | 「潮の作法だ。通れば、整う」「**玄界灘へ行け。沖ノ島は、通れ**」 |

#### 寄港せぬはずの着岸

| 段 | 内容 |
|----|------|
| **異変** | 彦島の後なのに左舷に島。霧が裂け**沖ノ島**——寄るつもりはなかった |
| **潮** | 舵が効かない。島へ吸い込まれる。「**導かれてる**」 |
| **祈祷の失效** | 親方が祝詞を唱えようとするが**音が出ない**——宗像側の作法が届かない |
| **言仁** | 動じない。「**通れ、と言った**」——招待でもあるように滑り込む |

#### 供物の結節（神託の前触れ）

| 項目 | 内容 |
|------|------|
| **指示** | 「**いちばんいいものを、海へ**」 |
| **実際** | 潮塩を岸へ流す。網の端を一尺切って帆に結ぶ。**金や米は捨てない** |
| **教経** | 「いちばんいいもの、と言いながら、いちばん捨てないもの。女神には形、俺たちには腹」 |
| **効果** | 大げさな供物ではないが、島の空気が変わる——**風が止む** |

> 第2話の解体・分配の続き。**形と腹**——結節と共同体の論理が、神域でも同じ。

#### 田心姫の神託（本話の核）
