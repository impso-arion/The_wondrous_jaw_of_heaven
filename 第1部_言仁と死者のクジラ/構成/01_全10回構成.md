## 巻全体の起承転結（本文準拠）

### 起（1〜2話）——海に乗り、天国を知る

| 段 | 内容 |
|----|------|
| **状況** | 米目当ての孤児・竹造が捕鯨船に乗る。「バカな話」の海と、同い年の異物・言仁（銛長） |
| **出来事** | 潮合・尖刺で初めてのクジラ。網を引く役割。岬の内の村へ——解体の共同体 |
| **竹崎** | 居場所なし→**天国**（米・クジラ・言仁の横）。郎党拒否されても離れない。**通訳の芽**（曲がった言葉を届ける） |
| **テーマ** | 楽園の発見。まだ組織の壁は遠い |

### 承（3〜5話）——警告を破り、仲間と宴を知る

| 段 | 内容 |
|----|------|
| **状況** | 彦島商い・玄界灘の漁。沖ノ島で**田心姫の神託**（これ以上取るな） |
| **出来事** | 神託を**破る**。木花（女船長）合流・共同捕鯨。宴のあと**宗像／原田方の侍**が乱入→頼舟が制圧 |
| **竹崎** | 捕鯨の流れを体で覚える。木花・頼舟・教経の輪。凪の日に**打毬**と**頼舟への特訓**（速さ・前に動く） |
| **テーマ** | 楽園は拡がるが、**正規（宗像・大宰府）の目**が近づく |

### 転（6〜8話）——死者の海へ、母へ

| 段 | 内容 |
|----|------|
| **状況** | 原田が死者のクジラを探す噂。定信（ジサイ）が**油・隔離・幽霊船**を説明 |
| **出来事** | **原田より先に**七日の彷徨。不知火・四艘（言仁木花／宗像／五島網組）。入り江で**死者のクジラ**に入る |
| **中** | 物部流の説明（言仁も全部はわからない）。**母**と対面。安徳の名・崇徳の**枝**・骨片。「**お前自身の国を、お前の足で**」 |
| **竹崎** | 通訳席・網・「押さえろ／聞け」。絆は深くないが、ついていく。外では**白旗・大船団** |
| **テーマ** | 楽園はもう海だけでは済まない。**載り・恨み・天子**の匂い |

### 結（9〜10話）——降伏と離散、姓だけが残る

| 段 | 内容 |
|----|------|
| **状況** | クジラ内で**崇徳分霊（勺）**との戦い。枝を切るが本体は別。浮上後、**原田種直**本人 |
| **出来事** | 監房→岩門城。**安徳／言仁**の正体が明かされる。言仁「降伏・人質」。各国へ**バラバラに送られる** |
| **結末** | 言仁のみ**宇都宮**の姓。竹造は**肥後**（通訳・手紙）。厳島学園へ（第2部） |
| **テーマ** | **組織に勝てない**。恭順＝計算された生存。宇都宮＝空いた殻の名字 |
