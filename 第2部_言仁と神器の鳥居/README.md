## 舞台の原則（厳守）

> **第2部は厳島・四魂の学舎のみ。外部政治は本編に入れない。**

| 書いてよい | 書かない |
|------------|----------|
| 学園内の人間関係・魔術・崇徳 | 豊前の領地会議 |
| **宇都宮**という肩書・殿様偏見 | 朝綱との直接対面 |
| 宇佐からの手紙一行 | 宇都宮軍の動員 |
| 北条・木花・竹崎の学園ドラマ | 宇佐神宮の内部政治 |

※ 宇都宮の軍勢は**第2部の舞台外**で育つ。第3部で初めて描く。
