## 生贄と「保留」

### 基本ルール

- 卒業生贄は**黄泉の国に行かない**。  
- **現世に留まり続ける**＝**保留状態**（崇徳の怨載り・鳥居増幅 → `../../00_シリーズ共通/世界観/03_崇徳院と怨載り.md`）。  
- 鳥居の下（裏側）に**留められている**。

### 人数と儀式

| 項目 | 内容 |
|------|------|
| **完全起動** | **12人**必要 |
| **すでに** | **10人**（崇徳のいけにえ。大潮の夜に鳥居下で会える） |
| **11人目** | **木花**（依代リスト上位・誘拐） |
| **12人目** | **言仁＝安徳**（崇徳が判明後に**歓喜**） |

### 大潮の夜

- 船で鳥居を**くぐる**と、保留中の**10人**に会える（話しかけてこない／ただ見ている、等）。  
- ホラー寄りだが、**講義しない**。竹崎の肌感のみ。

## 崇徳の野望と「未完成起動」（第10話）

### 校長が安徳と知ったとき

- **歓喜**する。  
- 12人目は「普通の生贄」ではなく、**安徳そのものへの受肉**。  
- 草薙がなくても、**体が国**——崇徳の解釈。

### 術の未完成

| 想定 | 実際 |
|------|------|
| 12人の保留が揃い、儀式完成 | 11人目（木花）＋12人目（受肉対象）で**型が噛み合わない** |
| 崇徳の完全復活 | **未完成のまま発動**。野望は**潰れかけ** |

### 第2部末の空気

- やや**気楽**：「黒幕、勝ち確の顔して**自爆**じゃねえか」（竹崎）  
- **木花は死なない**（術未完成・乱闘・出奔で**保留**）  
- 崇徳は**完全敗北ではない** → 第3部でパージ

### 言仁

- 崇徳の歓喜を知っても**動揺しない**（または最初から穴を読んでいた、程度の不可解さ）。  
- 受肉を**拒否する材料**として扱う。草薙と同様、**神器＝自分**ではない。  
- 第3部：**自ら神器たるしるしを抜き**、人として生きる（→ 第3部 `設定/01_言仁の結末と神器たるしるし.md`）
