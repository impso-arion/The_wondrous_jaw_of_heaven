#### 十如是（奇魂の理論核）

> 授業では**比喩**として語れる（講義録・全文NG）。本文は**アサガオ**で十分。

| 位相 | 比喩 | 奇魂での読み |
|------|------|-------------|
| **因（いん）** | アサガオの**種** | 祝詞の**核**・言いなおしの起点。なければ何も始まらない |
| **縁（えん）** | 土・水・光・**育てる手** | 結界・潮流・師弟・**今の位相**。因だけでは芽は出ない |
| **果（か）** | 内部で**次の種**の準備 | 祝詞の**裏**が組み上がる段階。まだ表に出ない |
| **報（ほう）** | **花が咲き、実が実る** | 潮・依代・式の**目に見える結果** |
| **本末究竟等** | 最初から最後まで**一つの真理** | 因→報は**バラバラではない**。位相を**飛ばす**と奇魂理論は**破綻** |

**奇魂の前提**：「**今、何をすべきフェーズか**」——**前後**を含めて読めなければ、祝詞の構造も結界の縫い目も**破綻**する。

#### 性格

| 軸 | 内容 |
|----|------|
| **表** | 細い声。目が**読む**。正規祝詞の型を**厳しく**教える。授業では**長く**説くこともある |
| **裏** | 結界の**縫い目**を知っている。校長を**評価する側**だが、これまで確信が持てなかった |
| **竹造** | 目的論・「今すぐ成果」「俺の主人は言仁」型の**位相の飛ばし**を**若気の至り**と感じる（説教口調にはしない） |
| **転換** | 第8話——竹造の**因**（論バトルの一言）＋崇徳の**怒り**で、校長＝**大魔縁**と**確信**。**報**（教師協力・追撃路）まで**最初にすべて理解する** |

#### 話し方

- **文**：授業では**長く**てよい（因縁の鎖・比喩）。本番・要請は**短く**切る
- **口癖**：「今は、**因**の位相だ」「祝詞は、隙から読む」「——見届けた」
- **NG**：崇徳同款の**演説**・**目的論すり替え**・内面独白
- **例（授業）**：「種だけ見て、花を急ぐな。——縁がない因は、芽にならない」
- **例（第8話末）**：「……**大魔縁**だ。協力を頼む。式を止め、道を開けてくれ」
- **例（因→報）**：「あの子の一言が、**因**だった。——**報**は、これからだ」（独白NG。態度・要請で示す）

#### 授業の芯（依代・二段の問い）

> 玉垣の教えは**安心**と**問い**が**セット**。説教口調NG。**丁寧な国学口調**。

**一段——神は何にでも宿る**

```
心配せずとも、神はなににでも宿る。
食事に使った箸に、きみたちが何も考えずに拾った石にさえ、神は宿ってくださる。
```

**二段——それでいいのか**

```
——それでいいのかい？
きみたちは、大切な神様を、そんなに簡単に宿っていただいていいのかい？

神様をお迎えするならば、その神様にふさわしい、
自分にとって大切なものをつくり、持ちなさい。
```

| 層 | 意味 |
|----|------|
| **表** | 依代は**身近**——誰でも祈れる（木花・ジサイの路線と**共鳴**） |
| **裏** | **安易な依代**＝崇徳の**序列・在庫**思考への**静かな反論** |
| **第5話** | 依代審の**前**または**後**——木花が**ジサイ＋自分の祈祷**で勝つ意味を、理屈ではなく**型**で示す |
| **第7話** | 演武庭の**依代石**——「何も考えず拾った石」にも神は宿る、が、木花の石だけ**答えが違う** |

**聞き手の反応（任意）**

| 誰 | 反応 |
|----|------|
| **木花** | 一段はほっとする。二段で**じいじ**を思い出す（口にしない） |
| **竹造** | 「……堅いな」（短く）。後で**因**として効く |
| **言仁** | 黙る。祝詞の**裏**と重なる |
